# 归档区

本目录包含已弃用、备份和临时数据。

## 目录结构

```
archive/
├── deprecated/              # 已弃用代码/文件
├── backup/                  # 历史备份
└── tempdata/                # 临时数据（定期清理）
```

## 说明

- **deprecated/**: 曾经使用但不再需要的代码和文件
- **backup/**: 重要文件的历史备份
- **tempdata/**: 实验产生的临时数据，建议每季度清理一次
