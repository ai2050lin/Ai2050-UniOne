# 第 61 章：WikiText-2 几何涌现长时实验 (Phase 3 Scaling)

**研究时间**：2026-02-19

**实验目标**：在更大规模数据 (WikiText-2) 和更深网络 (20M Params) 上验证 FiberNet 的几何涌现特性，重点观测内在维度 (ID) 的长时演化。

**实验配置 (Hyperparameters)**：
*   **模型**: ScaledFiberNet (20M)
    *   d_model=384, n_layers=12, n_heads=12, d_ff=1536
    *   **架构**: Logic-Memory 解耦 (Split Stream) + Geodesic Attention
*   **数据**: WikiText-2 (10M tokens)
*   **训练**: Batch=64, SeqLen=128, Steps/Epoch=1250 (Approx. 1 epoch pass)

**实验结果 (Epoch 1-14)**：
1.  **几何压缩期 (Compression Phase, Ep 7-11)**：
    *   内在维度 (ID) 从初始化状态迅速下降至 **~10.5**。
    *   **分析**：模型正在快速压缩冗余自由度，捕捉数据的低维核心流形 (Core Manifold)。
2.  **结构重组与膨胀期 (Reorganization & Expansion, Ep 12-14)**：
    *   ID 出现显著反弹：**23.8 (Ep 12) -> 25.4 (Ep 13) -> 27.2 (Ep 14)**。
    *   **Loss** 持续下降 (0.89 -> 0.85)。
    *   **分析**：这并非过拟合，而是典型的 "Grokking" 前兆或相变特征。模型在掌握核心结构后，开始利用额外的维度来编码更细粒度的特征 (Feature Split)，或者正在经历流形的全局重拓扑 (Re-topology)。

**核心发现**：
*   **动态维度呼吸 (Dimensional Breathing)**：智能系统的几何复杂度并非单调递减，而是呈现“压缩-寻找核心-再膨胀-精细化”的呼吸律动。
*   **Memory Stream 敏感性**：几何观测必须基于 Memory Stream (Index 2)。Logic Stream (Index 1) 倾向于维持通过性的恒等映射，而 Memory Stream 才是承载数据拓扑变化的真实载体。

**结论**：Phase 3 成功捕捉到了大模型训练过程中的非微凡几何动态。FiberNet 展现出了符合 SHMC 理论预期的流形自组织行为。
