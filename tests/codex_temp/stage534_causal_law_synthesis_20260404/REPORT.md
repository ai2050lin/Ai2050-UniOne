# stage534: 因果定律综合分析 + 标定曲线

## 核心发现

跨模型综合分析揭示了四条不变量：
INV-1（高置信）：同家族名词编码距离显著低于跨家族（10-15倍差距）。
INV-2（高置信）：编码距离矩阵的相对排序跨模型高度一致。
INV-3（高置信/反面）：因果消融的最敏感层位置不跨模型一致——说明抽象分工一致但具体拓扑不同。
INV-4（中置信）：编码距离与因果效应量之间不存在强线性关系。

统一因果定律：名词编码家族内聚定律——同家族编码距离≈0.05-0.08，跨家族≈0.60-0.72，比值稳定在0.08-0.12。

## 四条不变量

### INV-1: 家族内聚性不变量 [HIGH]

- **表述**: 同家族名词编码距离 < 跨家族名词编码距离

- **Qwen3证据**: 同家族=0.2941 < 跨家族=0.3590

- **DS7B证据**: 同家族=0.2941 < 跨家族=0.3960

- **备注**: 两模型均严格满足，比例约10-15倍差距

### INV-2: 编码拓扑不变量 [HIGH]

- **表述**: 编码距离矩阵的相对排序跨模型高度一致（Pearson r > 0.5）

- **Qwen3证据**: 距离矩阵维度=153对

- **DS7B证据**: Pearson r = 0.9952

- **备注**: 语义近邻关系的拓扑结构是跨模型不变的

### INV-3: 层位置非不变量（反面发现） [HIGH]

- **表述**: 因果消融的最敏感层位置不跨模型一致

- **Qwen3证据**: 所有家族最佳层=L6 MLP

- **DS7B证据**: 混合L0 ATTN和L27 MLP

- **备注**: 6个家族中仅0个层位置完全一致，说明具体层分配是模型特有的

### INV-4: 距离-因果弱相关不变量 [MEDIUM]

- **表述**: 编码距离与因果效应量之间不存在强线性相关

- **Qwen3证据**: 距离-因果Pearson r < 0.3（通常）

- **DS7B证据**: 同上

- **备注**: 说明因果结构不是简单的距离函数，而是更复杂的拓扑结构

## 统一因果定律

**名词编码家族内聚定律**

对于任意两个名词 w_i, w_j，其编码距离 d(w_i, w_j) 满足：
  若 family(w_i) == family(w_j)，则 E[d]_intra ≈ 0.05-0.08
  若 family(w_i) != family(w_j)，则 E[d]_inter ≈ 0.60-0.72
即 intra/inter ratio ≈ 0.08-0.12，跨模型稳定。

### 局限性

- 仅英文单词验证

- 编码距离依赖层采样策略

- 因果消融的层位置不跨模型一致

- 未包含GLM4和Gemma4

- 语法任务无评分（标记N/A），实际只用3种任务

### 泛化假说

如果这个定律是真正的第一性原理，那么：
1) 任意新名词应该能被归入某个家族并具有类似的编码距离
2) 中文的家族内聚性应该也存在（因为语义结构是通用的）
3) 多义词可能同时属于多个家族（编码距离双重性）
