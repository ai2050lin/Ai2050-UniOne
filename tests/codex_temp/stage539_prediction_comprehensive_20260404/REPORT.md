# stage539+540: 预测定律综合验证

## 核心发现

预测定律验证揭示了编码结构的深层特征：
1) 留一法预测优势极弱（Qwen3=0.99x, DS7B=0.94x），说明单token编码中家族信息占比不大；
2) 全局骨干是主导分量（Qwen3=0.861, DS7B=0.713），所有名词共享大部分编码；
3) 家族骨干与独有残差近似正交（Qwen3 cos=-0.057, DS7B cos=-0.070），三分量独立分解成立；
4) 但留一法预测优势仅1.0x说明：在单token级别，家族信息被全局信息淹没。需要在完整句式上下文中才能检测到家族效应。
