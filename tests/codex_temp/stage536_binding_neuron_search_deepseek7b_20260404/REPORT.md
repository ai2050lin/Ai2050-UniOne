# stage536: 绑定瓶颈神经元搜索 (DeepSeek7B)

## 核心发现

DeepSeek7B上的绑定项研究复现了Qwen3的主要发现，并新增了神经元级分析：
1) 属性绑定仍然是效率最高的绑定类型（信息集中度最高）；
2) 同类型绑定的神经元Jaccard相似度可量化绑定神经元的共享程度；
3) 跨绑定类型的神经元交集较小，说明不同绑定类型使用不同的神经元子集。
