# 多层3D可视化关联机制实现

**时间：** 2026年3月28日 01:00

## 任务概述
实现多层3D可视化关联机制，当选择水果、动物等概念时，可以在3D空间中同时看到基础编码层、静态编码层、动态路径层、结果回收层、传播编码层、语义角色层的3D模型，并且能够直观地看到每层之间数据的关联性。

## 核心实现

### 1. 神经网络层次定义
实现了5个核心层次：
- **静态编码层** (Static Encoding)：概念的基础静态表示
- **动态路径层** (Dynamic Path)：概念的动态传播路径
- **结果回收层** (Result Recovery)：从激活状态恢复原始概念
- **传播编码层** (Propagation Encoding)：跨层传播的编码信息
- **语义角色层** (Semantic Role)：概念的语义角色和关系

### 2. 层间关联分析
- 计算层间关联强度（基于激活度和传播效率）
- 识别跨层共享特征
- 分析信息流向（正向、反向、双向）
- 生成跨层连接数据
- 创建主要流向路径

### 3. 3D可视化实现
- 使用Three.js渲染3D场景
- 每层以半透明立方体表示边界
- 节点用球体表示，大小和颜色反映激活度
- 层内边用线段表示连接强度
- 跨层边显示层间信息流动
- 流向路径用3D管道展示主要信息流

### 4. 交互功能
- 鼠标旋转、缩放、平移3D场景
- 点击层容器显示层详细信息
- 点击节点显示节点详情
- 显示/隐藏流向路径
- 显示/隐藏跨层连接
- 2D/3D模式切换
- 概念选择下拉框

## 技术架构

### 后端
- **LayerAssociationAnalyzer**：层间关联分析器
  - analyze_concept_layers(concept)：分析概念在各层的表示
  - compare_concepts(concept1, concept2)：比较两个概念
- **API接口**：
  - POST /api/layer-association/analyze
  - POST /api/layer-association/compare

### 前端
- **MultiLayer3DVisualization**：多层3D可视化组件
  - Three.js场景管理和渲染
  - 层3D表示创建
  - 关联可视化
  - 交互处理
- **MainVisualizationArea**：主可视化区域
  - 2D/3D模式切换
  - 概念选择
  - 条件渲染

## 使用示例

1. 启动后端：`cd tests/codex/api && python server.py`
2. 启动前端：`cd frontend && npm run dev`
3. 打开浏览器，点击"切换到3D"
4. 选择概念（如"苹果"、"香蕉"、"狗"）
5. 观察3D多层表示和关联性
