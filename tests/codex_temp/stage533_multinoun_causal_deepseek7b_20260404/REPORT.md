# stage533: 多名词家族统一因果搜索 (DeepSeek7B)

## 核心发现

DeepSeek7B上的六类名词家族因果搜索揭示了跨模型一致性：1) 同家族名词编码距离显著低于跨家族；2) 因果消融的层位置可能与Qwen3不同但抽象分工一致；3) 标定数据已生成，可与Qwen3对比。
