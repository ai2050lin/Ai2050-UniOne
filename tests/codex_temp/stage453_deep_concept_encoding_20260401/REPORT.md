## 关键发现与理论修正

### 发现1: 分层编码机制
- 不同层对概念编码的贡献差异巨大
- 存在'概念特异性层'：这些层的R²远高于简单平均
- 低层可能编码基础属性，高层编码抽象语义

### 发现2: 概念算术的数学结构
- 同类别概念算术（苹果→香蕉）比跨类别（苹果→猫）更有效
- 偏置转移在特定语义维度上保持方向一致性
- 加权偏置方法优于简单平均方法

### 发现3: 属性编码的层级分布
- 颜色、大小、味道等属性在特定层集中编码
- 属性编码层与概念区分层部分重叠但不完全一致

### 修正后的编码方程
```
概念编码 = W_layer * (B_category + a_individual)
  其中:
  B_category = 类别基底（跨层加权组合）
  a_individual = 个体偏置（稀疏向量，约34%维度活跃）
  W_layer = 层权重（不同层对概念编码的贡献不同）
```

### 瓶颈与下一步
1. R²仍然偏低（~0.4），需要更精细的基底构建方法
2. 概念算术的跨类表现不稳定，需要理解'语义空间'的几何结构
3. 需要Stage454：使用SVD/ICA分解基底，发现独立语义因子
