# stage535: 绑定项互信息量化 (Qwen3)

## 核心发现

四类绑定类型的逐层信息追踪揭示了：
1) 每种绑定类型都有各自的'绑定瓶颈层'——信息绑定最集中的层；
2) 属性绑定和关系绑定的信息注入量最大，语法绑定最小；
3) 消融瓶颈层可因果性地削弱绑定强度，证明绑定是真实的计算过程而非巧合。
