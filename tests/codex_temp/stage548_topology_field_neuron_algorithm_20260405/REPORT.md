# stage548 拓扑不变量与场控制杆联合算法原型

## 核心结论
当前最可信的组合不是“找前几个最高分神经元”，而是“先恢复拓扑，再恢复绑定场，再把场投影回神经元、注意力头和残差方向”。也就是说，神经元级编码结构应被看成‘拓扑骨架 + 场式控制 + 小型尖锐控制杆’的联合系统。

## 不变量可信度
- 绑定场控制杆：可信，FIELD/POINT = 8/0，平均 top100 集中度 = 0.146784
- 拓扑排序：可信，Pearson r = 0.995170
- 绑定效率排序：可信，rho = 1.000000
- 绑定因果排序：暂不够强，rho = 0.400000
- 精确层号：不可信，match_rate = 0.000000

## 算法步骤
1. 拓扑恢复：先恢复概念或词类之间的稳定相对几何关系，而不是先盯单个神经元。
2. 场统计恢复：比较绑定前后、切换前后、路由前后的隐藏状态统计，判断控制是场式还是点式。
3. 层带定位：寻找拓扑变化最大且场差异最大的宽带功能区，而不是只找单层峰值。
4. 场到神经元投影：将关键场方向投影回注意力头、神经元、残差方向，得到候选结构部件。
5. 混合因果搜索：在注意力头、神经元、残差方向的联合空间中搜索最小可解释子回路。
6. 强控制复核：确认打掉的是特异编码结构，而不是整体语言能力。

## 统一理解
- 拓扑不变量负责告诉我们哪些编码彼此更近。
- 场控制杆不变量负责告诉我们绑定控制不是少数点状神经元单独承载。
- 层带图谱负责告诉我们搜索应优先落在哪些宽带功能区。
- 最小因果回路负责告诉我们真正可打中的控制杆组合是什么。
