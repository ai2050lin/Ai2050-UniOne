## 分能力稳定性

### 共同趋势

- `syntax / relation / coref` 在 `qwen3`、`deepseek7b`、`glm4` 上整体稳定。
- `style` 在四模型上都弱，说明当前“单词候选风格协议”还不足以代表真正的风格编码。
- `disamb` 在 `qwen3` 与 `deepseek7b` 上只有半稳定，在 `glm4` 上最稳定，在 `gemma4` 上失败。

### 分模型观察

- `qwen3`
  - 总体稳定。
  - `disamb` 半稳定，`style` 明显失败。
  - `syntax` 的末层绝对对齐最高。

- `deepseek7b`
  - 整体和 `qwen3` 接近，但绝对对齐更低。
  - 峰值层集中在 `27`，明显早于 `qwen3` 的 `35`。

- `gemma4`
  - 总体表现最弱。
  - `d_norm_growth` 极低，仅 `4.22`。
  - 旋转角最高，`52.91°`，与此前“高速旋转 + 消歧差”的旧结论一致。
  - 峰值层集中在 `16-19`，明显偏早且后续没有持续放大。

- `glm4`
  - 总体最好，`overall_pair_accuracy = 0.80`。
  - `disamb / syntax / relation / coref` 全部稳定，只有 `style` 继续失败。
  - `d_norm_growth` 爆发式增大到 `6870.73`，远高于其他模型。
  - 峰值层集中在末层 `39`，符合“持续放大”型策略。

## 第一轮最重要的理论结论

1. “差异方向动力学”并不只出现在多义消歧里。
   - 在语法、关系、指代上，至少 `qwen3 / deepseek7b / glm4` 都出现了明显放大与逐层旋转。

2. 四模型确实存在明显的编码策略差异。
   - `glm4` 偏“持续放大”
   - `gemma4` 偏“早峰值 + 强旋转 + 低放大”
   - `qwen3 / deepseek7b` 处于中间

3. “风格”目前不是同一协议下的稳定能力。
   - 这不是坏事，反而说明当前统一理论还没有偷懒地把所有能力都装进去。

4. 统一理论的第一版硬边界已经出现。
   - 如果一个能力在四模型上始终弱且不稳定，就不能强行说它已经被当前变量体系解释。

## 当前最严格的表述

- 现在可以更谨慎但更有把握地说：
  - `h_l / delta_l / d_l` 这组变量不再只是“消歧专用变量”，它们已经开始对语法、关系、指代显示出跨能力解释力。
- 但同样必须承认：
  - 这套变量体系还没有覆盖风格控制。
  - 它目前更接近“离散判别型语言能力动力学”，还不是完整语言统一理论。

## 下一步建议

1. 扩充 `style` 协议
   - 不再只用单词候选
   - 改为短句改写对比、礼貌等级、正式度分级

2. 扩充 `disamb` 样本
   - 每类至少扩到 `5` 对
   - 降低单个提示模板偏置

3. 做第二轮跨模型验证
   - 保持现有四模型
   - 但把每类样本数提升后再看不变量是否仍成立

4. 加入中层读出分析
   - 当前峰值层几乎总落在末层，说明“只看最后 token”过于粗糙
   - 下一轮应加入中层对齐和中层 margin 代理指标
