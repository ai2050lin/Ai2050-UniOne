## 第二轮核心观察

### 1. 三强一弱格局依旧存在

- `qwen3 / deepseek7b / glm4` 继续构成“可用组”。
- `gemma4` 继续明显落后。

### 2. 共享规律更清晰了

- `syntax` 在 `qwen3 / deepseek7b / glm4` 上都达到 `1.0`。
- `disamb` 在 `qwen3 / deepseek7b / glm4` 上都达到 `0.8`。
- `relation` 在 `qwen3` 与 `glm4` 达到 `1.0`，`deepseek7b` 为 `0.8`。
- `style` 虽然仍弱，但已经不再是完全失效：
  - `qwen3 / deepseek7b / glm4` 都到 `0.4`
  - 说明风格协议改进后，至少开始捕捉到部分真实信号

### 3. Gemma4 的异常不是偶然

- `gemma4`
  - `d_norm_growth = 3.48`
  - `rotation_deg = 52.90`
  - `syntax pair_accuracy = 0.0`
  - `disamb pair_accuracy = 0.0`
- 这与之前“高旋转、低放大、早峰值”的旧结论高度一致。
- 说明其弱点不是首轮小样本造成的噪声，而是更稳定的编码风格特征。

### 4. GLM4 的持续放大策略仍最强

- `glm4`
  - `pair_accuracy = 0.80`
  - `d_norm_growth = 5848.25`
  - `peak_layer = 39`
- 它在第二轮仍然维持“持续放大到末层”的策略。

### 5. 中层指标带来一个新发现

- `qwen3 / deepseek7b` 的 `mid_d_norm` 远高于 `glm4`：
  - `qwen3 = 21.00`
  - `deepseek7b = 103.63`
  - `glm4 = 5.05`
- 但 `glm4` 最终 `d_norm_growth` 最大。
- 这说明：
  - `glm4` 的优势不在“中层已经很强”
  - 而在“后层持续放大能力”更强

## 按能力看

### syntax

- 最稳定能力之一。
- `qwen3 / deepseek7b / glm4` 都为 `1.0`。
- `gemma4` 为 `0.0`。
- 说明“语法一致性”非常适合作为共享变量体系的稳定测试任务。

### disamb

- 第二轮显著变稳。
- `qwen3 / deepseek7b / glm4` 都达到 `0.8`。
- `gemma4` 仍为 `0.0`。
- 说明扩样本后，“消歧半稳定”已经提升为“消歧基本稳定”。

### relation

- `qwen3 / glm4` 最稳，`deepseek7b` 次之。
- `gemma4` 仍偏弱。

### coref

- `deepseek7b / glm4` 达到 `0.8`，`qwen3 / gemma4` 为 `0.6`。
- 指代比语法和关系更难，但仍能被当前变量体系部分抓住。

### style

- 仍然是当前体系的最弱环节。
- 但从第一轮的“几乎完全失效”变成了第二轮在三模型上 `0.4`。
- 这说明风格不一定无法纳入，而是协议还没有打磨到位。

## 第二轮最严格的理论表述

- 现在可以比第一轮更有把握地说：
  - `h_l / delta_l / d_l` 这组变量，已经对多义消歧、语法一致性、关系抽取、部分指代消解表现出跨能力解释力。
- 但还不能说：
  - 它们已经统一解释了完整语言能力。
- 更严谨的说法是：
  - **这套变量体系目前最适合刻画“离散判别型语言能力”的编码动力学。**

3. 做组件级验证
   - 既然第二轮已经说明变量跨能力存在
   - 下一步应在 `syntax / relation / coref` 上做 `attn / mlp` 零化与注入

   - 把“已被解释能力”和“尚未被解释能力”明确分开
   - 避免理论叙述过度外推
