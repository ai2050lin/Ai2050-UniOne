# stage519 名词-属性-桥接层带图谱

## 核心结论
名词、属性、桥接并不是完全均匀散在所有层里，而是通常呈现宽带分布中的明显层带偏置。也就是说，它们既不是单层点状，也不是全层均匀噪声，而更像若干相对稳定的层区。
