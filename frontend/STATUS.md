### 使用说明

#### 启动前端
```bash
cd d:\develop\TransformerLens-main\frontend
npm run dev
```

#### 访问应用
在浏览器中打开: http://localhost:5173

#### 使用DNN分析功能
1. 导航到"语言研究"控制面板
2. 展开"DNN分析"区域
3. 选择一个分析维度
4. 点击"运行分析"按钮
5. 等待分析完成（使用模拟数据）
6. 点击"切换到3D"按钮查看3D可视化
7. 使用鼠标交互探索3D视图
