### 硬伤 #2: 多步推理能力未验证

**问题描述**:
- Grokking 测试仅涉及单步运算 (a + b)
- 未测试复杂多步推理 (如数学证明、代码生成)
- 未验证长期依赖关系的处理

**影响**: 无法处理 AGI 级别的复杂任务

**证据**:
```
当前测试:
- 群运算: 1步
- 类比: 1步
- 推理链长度: 未测试

AGI 需求:
- 数学证明: 10+ 步
- 代码生成: 20+ 步
- 长期规划: 100+ 步
```

**缓解方案**:
1. 设计多步推理基准测试
2. 测试思维链 (Chain-of-Thought) 能力
3. 验证长上下文推理

**风险等级**: 🔴 高
