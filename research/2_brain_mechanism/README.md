# 路线2: 大脑机制还原

## 研究目标

通过神经科学实验数据，验证DNN分析得出的编码机制假说，还原大脑真实的编码原理。

## 核心问题

1. DNN发现的稀疏编码在大脑中是否存在？
2. 大脑的能耗约束如何塑造编码结构？
3. 局部Hebb学习如何产生全局编码结构？

## 核心假说

### 假说1: 稀疏编码

```
预测: 大脑通过神经元放电阈值+GABA侧向抑制实现稀疏编码
验证: 阻断GABA受体 → 稀疏度下降 → 编码混淆
数据: V1区简单细胞记录
```

### 假说2: 高维编码

```
预测: 大脑利用千亿神经元的天然正交性编码概念
验证: 概念数量增加 → 激活模式越来越正交
数据: fMRI多体素模式分析
```

### 假说3: 能效驱动

```
预测: 20W功耗约束是稀疏编码进化的驱动力
验证: 节能环境训练的模型 → 更稀疏编码
数据: 大脑发育数据 + 计算实验
```

### 假说4: 局部涌现

```
预测: Hebb学习+突触可塑性 → 特征涌现
验证: 学习关键期 → 编码结构急剧变化
数据: 发育神经科学数据
```

### 对比分析

```
DNN激活 ←→ 大脑激活
  ├── RSA（表征相似性分析）
  ├── 激活模式对比
  └── 稀疏度对比
```

## 验证实验设计

### 实验1: 稀疏编码验证

**方法**: RSA对比DNN和大脑激活
**预测**: 相似任务的激活模式相似
**成功标准**: RSA相关系数 > 0.5

### 实验2: GABA阻断实验

**方法**: 药理学实验（需合作）
**预测**: 阻断抑制 → 稀疏度下降
**成功标准**: 稀疏度变化显著

### 实验3: 能效编码验证

**方法**: 计算实验 + 大脑数据对比
**预测**: 节能环境 → 更稀疏编码
**成功标准**: 一致性 > 0.7
