# 路线1: DNN结构分析

## 研究目标

从训练好的深度神经网络中提取特征编码结构，分析其数学性质，为理解大脑编码机制提供参照。

## 核心问题

1. DNN内部形成了什么样的特征编码？
2. 特征如何在层级间演化和抽象？
3. 稀疏编码、高维编码等模式是否存在？

## 关键发现

### 1. 稀疏编码存在

```
所有层L0稀疏度稳定在~78%
接近大脑~2%神经元同时激活的稀疏模式
推断: 大脑可能通过神经元阈值+GABA抑制实现稀疏编码
```

### 2. 层级特征演化

```
抽象比率: Layer 0 (1.01) → Layer 11 (1.11)
精确度: Layer 0 (40%) → Layer 11 (80%)
内在维度: Layer 6最低(4.28)，信息最压缩
```

### 3. 正交性特征

```
所有层正交性 > 0.97
符合高维空间天然正交的预期
支持"高维编码"假说
```
