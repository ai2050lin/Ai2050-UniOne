#### 数学形式化

**编码方程**:
\\[
h_l = \\sigma(W_l h_{l-1} + b_l) \\odot m_l
\\]

其中：
- \\( h_l \\): 第l层的隐藏状态
- \\( \\sigma \\): 非线性激活函数
- \\( W_l, b_l \\): 第l层的权重和偏置
- \\( m_l \\): 稀疏掩码，19%元素为1

**稀疏性约束**:
\\[
\\sum_{i=1}^{n} m_l[i] = 0.19 \\times n
\\]

**分布式编码约束**:
\\[
\\frac{1}{|C|} \\sum_{c', c'' \\in C} \\text{sim}(h(c'), h(c'')) \\approx 0
\\]

**跨层独立编码约束**:
\\[
\\text{shared}(h_l, h_{l'}) = 0 \\quad \\text{for} \\quad l \\neq l'
\\]
