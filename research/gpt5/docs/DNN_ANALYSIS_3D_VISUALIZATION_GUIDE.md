# DNN分析3D可视化功能说明

## 添加时间
2026年3月28日 23:55

## 功能概述

为DNN分析控制面板添加了完整的3D可视化功能，用户可以在2D数据视图和3D空间可视化之间切换，提供更直观的分析体验。

## 新增功能

### 1. 3D可视化组件

#### 文件：`frontend/src/components/DNNAnalysis3DVisualization.jsx` (约400行)

**核心组件**：
- `NeuronNode3D` - 3D神经元节点，支持交互和激活度显示
- `Connection3D` - 3D连接线，支持动态流动效果
- `EncodingStructure3D` - 编码结构3D可视化
- `AttentionPattern3D` - 注意力模式3D可视化
- `FeatureExtraction3D` - 特征提取3D可视化
- `LayerDynamics3D` - 层间动力学3D可视化
- `NeuronGroups3D` - 神经元分组3D可视化

### 2. 3D可视化特性

#### 交互功能
- **鼠标悬停**：显示节点标签和激活度
- **点击交互**：触发节点点击回调
- **相机控制**：支持旋转、缩放、平移
- **动态效果**：连接线流动动画

#### 视觉效果
- **激活度指示**：颜色和高度反映激活程度
  - 绿色：高激活 (>0.7)
  - 黄色：中等激活 (0.4-0.7)
  - 红色：低激活 (<0.4)
- **连接强度**：线条粗细和透明度反映连接强度
- **动态粒子**：在注意力模式中显示信息流动

#### 不同维度的可视化风格

**1. 编码结构 (Encoding Structure)**
- 层状布局，显示神经元激活模式
- 层间连接显示信息流动
- 激活度通过颜色和高度可视化

**2. 注意力模式 (Attention Patterns)**
- 环形布局，强调层间注意力
- 主导层用大节点标识
- 动态连接显示注意力流动

**3. 特征提取 (Feature Extraction)**
- 簇状布局，显示特征分组
- 不同颜色代表不同特征簇
- 随机分布模拟真实特征空间

**4. 层间动力学 (Layer Dynamics)**
- 线性布局，显示信息传播
- 关键层突出显示
- 粒子流展示传播过程

**5. 神经元分组 (Neuron Groups)**
- 功能分区布局
- 不同角色用不同颜色
  - 绿色：编码
  - 蓝色：路由
  - 橙色：输出
  - 粉色：注意力
  - 紫色：记忆

### 3. 用户界面改进

- 移动端友好

## 使用方法

### 基本操作

1. **选择分析维度**：在"DNN分析维度"区域选择一个分析维度
2. **运行分析**：点击"运行分析"按钮
3. **查看2D结果**：分析完成后，默认显示2D数据表格
4. **切换到3D视图**：点击"3D视图"按钮
5. **交互探索**：
   - 鼠标拖动：旋转视角
   - 滚轮：缩放
   - 悬停节点：查看详细信息
   - 点击节点：触发回调

### 高级功能

1. **多维度对比**：可以同时运行多个分析维度，对比结果
2. **层范围调整**：调整层范围参数，观察不同深度的可视化
3. **模型切换**：切换不同模型，对比3D结构差异

## 性能考虑

### 节点数量控制
- 编码结构：每层最多50个神经元
- 注意力模式：最多5个主导层
- 特征提取：最多10个特征簇
- 层间动力学：最多4个关键层
- 神经元分组：最多12个分组

### 渲染优化
- 使用 `MeshDistortMaterial` 替代复杂几何体
- 合理控制连接数量
- 使用动画优化减少重绘

### 内存管理
- 组件卸载时清理资源
- 避免内存泄漏
- 合理使用 `useEffect`
