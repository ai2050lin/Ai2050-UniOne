## 2026-04-05 神经元级结构获取算法原型

对应结果：
- [stage548_topology_field_neuron_algorithm.py](/d:/develop/TransformerLens-main/tests/codex/stage548_topology_field_neuron_algorithm.py)
- [stage548 summary.json](/d:/develop/TransformerLens-main/tests/codex_temp/stage548_topology_field_neuron_algorithm_20260405/summary.json)

当前最值得保留的判断是：

1. `绑定场控制杆不变量` 可信，而且目前是最强不变量之一。  
   现有证据表明，绑定差异主要不是由少数“点状神经元”单独承载，而更像由一片分布式场承载，再叠加少量尖锐控制杆。

2. `拓扑排序不变量` 也可信。  
   跨家族编码的相对几何关系高度稳定，因此“谁和谁更近”的相对结构，比“精确落在第几层”更像真正可保留的机制约束。

3. 因此，神经元级结构获取不能再从“前几个最高分神经元”出发，而应从下面这条统一流水线出发：
   1. 先恢复概念或词类的拓扑排序。
   2. 再恢复绑定场、切换场、路由场的统计结构。
   3. 再定位拓扑变化最大、场差异最大的层带。
   4. 再把这些场方向投影回 `attention head（注意力头）`、`MLP neuron（多层感知机神经元）`、`residual direction（残差方向）`。
   5. 最后用最小混合因果回路搜索和强控制协议复核。

更通俗地说：

**先找概念地图，再找控制场，再把控制场拆成神经元回路。**

这意味着当前最合理的神经元级工作模型不再是：

- 一个概念对应一团固定神经元

而是：

- `拓扑骨架`
- `场式控制`
- `小型尖锐控制杆`

三者叠加后，构成最终可被因果打中的编码结构。

## 2026-04-05 名词家族神经元级结构提取协议

对应结果：

- [stage549 summary.json](/d:/develop/TransformerLens-main/tests/codex_temp/stage549_noun_family_neuron_structure_protocol_20260405/summary.json)

当前名词主线已经可以被压成更明确的结构模板：

`noun_encoding = global_backbone + family_backbone + noun_unique_residual + task_bridge_adapter + cross_task_causal_core`

也就是：

- `全局共享名词骨干`
- `家族共享骨干`
- `名词独有残差`
- `任务桥接适配器`
- `跨任务因果核心`

这意味着像 `apple（苹果）` 这样的名词，不再适合被理解成“一整块固定神经元表示”，而更像：

1. 先复用一部分所有名词都共享的底座；
2. 再复用一部分水果家族共享骨干；
3. 再叠加苹果自己的独有残差；
4. 再根据知识、语法、属性、联想任务挂上不同桥接适配器；
5. 最后由一小组跨任务因果核心把整套结构稳定下来。

当前四模型都已经能输出这五层结构字段，但共享厚度、独有残差厚度、桥接强度和因果核心厚度明显不同，这意味着：

**名词机制已经不只是“有没有共享骨干”的问题，而是“共享厚度、独有残差、桥接风格、因果核心强度”的分型问题。**

## 0. 文档定位

这份文档不是实验流水账，也不是某一次阶段汇报。
它的目标只有一个：

**系统化讲清楚当前对“深度神经网络语言编码机制”的研究进展，包括原则、目标、测试体系、核心拼图、当前结论、问题硬伤和下一阶段计划。**

当前最重要的背景判断是：

1. 深度神经网络已经提取了某种高度压缩的语言数学结构。
2. 这套结构既承载知识网络，也承载语法、风格、逻辑和概念关联。
3. 它很可能不是“词典式存储”，而是“分层控制系统”。
4. 现在的工作不是先发明一套漂亮理论，而是先通过大量拼图逼近真实结构。

## 1. 研究原则

### 1.1 先拼图，后理论

当前主线不是先写公理，而是先还原机制。
严格顺序应当是：

1. 先积累可复核的实验拼图。
2. 再压缩出稳定结构。
3. 再做因果打击和跨模型复核。
4. 最后才讨论是否有资格上升为第一性原理理论。

最核心的一句话是：

**先机制，后定理；先还原，后抽象。**

### 1.2 先因果，后修辞

一个结论想被保留，尽量要满足：

1. 能在原始激活或原始行为里直接看见。
2. 能被消融或干预打中，不只是相关。
3. 能跨样本复现。
4. 能跨模型复现。
5. 能被更强实验推翻或修正。

### 1.3 先统一变量，后局部故事

当前最危险的错误是：

1. 把局部亮点误当成全局机制。
2. 把工程噪声误当成理论信号。
3. 把单个模型风格误当成普遍定律。

因此当前研究始终优先寻找统一控制变量，而不是堆更多孤立个案。

---

## 2. 核心目标

当前总目标不是证明模型“会说话”，而是回答下面这些更硬的问题：

1. 语言信息在网络内部如何分层组织。
2. 名词、属性、关系、代词、功能词、词元补全是否共享同一套高层控制结构。
3. `attention head（注意力头）`、`MLP neuron（多层感知机神经元）`、`residual stream（残差流）` 分别承担什么角色。
4. 这些能力是稀疏小回路实现，还是分布式场实现。
5. 不同模型是否共享同一抽象机制，只是拓扑不同。
6. 能否把这些拼图压成统一状态方程。

## 4. 当前统一工作方程

截至目前，最稳的工作方程是：

`h_t^{l+1} = h_t^l + C_l(form_t, context_t) + R_l(ref_t, context_t) + B_l(lemma_t) + S_l(lemma_t, context_t) + A_l(attr_t, context_t) + G_l(B_l, A_l, C_l, R_l) + O_l(h_t^l)`

这不是最终定理，而是当前最稳的工作模型。

各项当前含义如下：

- `C_l`
  词形连续性控制项。负责把局部字形或前缀推到高概率后续词元。

- `R_l`
  路由项。负责代词、功能词、句法角色和跨位置依赖的路由。

- `B_l`
  概念骨干项。负责共享概念底座、家族位置和公共对象承载。

- `S_l`
  切换项。负责多义词在共享底座上的低成本切换。

- `A_l`
  属性项。负责颜色、味道、大小、功能、角色等修饰方向。

- `G_l`
  绑定项。负责把对象、属性、关系和当前实例拼起来。

- `O_l`
  读出项。负责把内部状态压成最终输出。

从更高层抽象看，当前最稳的分工仍是：

1. `attention head` 更像负责路由、搬运、配对、条件选择。
2. `MLP neuron` 更像负责局部写入、目标放大、落点锁定。
3. `residual stream` 更像负责共享工作记忆和层间传播。

3. 更合理的说法是：
   - 词类在最尖锐控制杆上高度分离
   - 词类在宽带功能区上仍共享更高层组织

---

## 6. 当前最稳结论

截至目前，最稳的结论可以压成下面八句：

1. 语言编码不像静态词典，更像分层控制系统。
2. `attention head`、`MLP neuron`、`residual stream` 是分工结构，不是同一种部件。
3. 功能词和代词已经露出路由骨架。
4. 多义词已经露出共享底座 + 切换结构。
5. 属性绑定已经露出对象骨干 + 属性通道 + 桥接区的结构定律。
6. 中文词元补全、功能词、固定搭配可以被压入同一图谱。
7. 同一个名词跨知识、语法、属性、联想任务确实复用同一部分骨干。
8. 跨任务最小因果子集已经开始在四模型中显形，说明概念骨干不是纯相关幻觉。
9. 有效神经元通常具有明显层带偏置，语言结构更像“宽带功能区”而不是均匀平铺。
10. 大量名词的编码已经露出“全局骨干 + 家族骨干 + 名词独有残差”的层级结构，而且这条结构已经开始支持家族级预测。
11. 把代词、动词、连接词、固定搭配、时间词、数量词、方位词纳入同一地图后，语言功能层区已经显形，不再只是名词与属性的局部现象。
12. 桥接因果已经从“名词-属性”扩展到“名词-关系”“名词-语法角色”“名词-联想网络”，说明概念骨干与任务接口之间的连接本身也可以被压成小型因果子集。
13. 把层带位置图谱与最小因果回路压到一起之后，当前最稳的统一动力学是：早层更偏路由，中层更偏绑定，晚层更偏收束读出。
14. 六类词性各自具有不同层带风格：名词更像对象骨干带，形容词更像修饰双峰带，动词更像动作带，副词更像调制带，代词和介词更像功能路由带。

如果只保留一句最稳的话，那就是：

**当前最接近事实的图景，是语言能力来自“共享概念骨干 + 路由机制 + 任务适配器 + 绑定机制 + 晚层读出”的分层控制系统，而不是一张静态词表。**

## 7. 问题、硬伤与瓶颈

当前理论还远没有闭环，硬伤很明确。

### 7.1 绑定项仍然最弱

`G_l` 目前仍以结构证据为主，强因果闭环不够硬。
虽然 `stage520` 已经把桥接推进到四模型第一轮因果比较，但整体强度仍明显弱于共享骨干主线。

### 7.2 阈值依赖仍然存在

很多复用数、差异数、交并比仍依赖 `top-k（顶部阈值）` 定义，不能误当自然常数。

### 7.3 GPU 仍不能单独充当最终真相来源

`CPU（中央处理器）` 和 `GPU（图形处理器）` 在趋势结论上通常能对齐，但在峰值层位、小信号和局部最强单元上仍会漂。

当前正式协议仍然是：

1. `GPU` 用于趋势级实验。
2. `CPU` 用于峰值层位、最强单元和细粒度方向耦合复核。

### 7.4 四模型并非同一种可观测深度

虽然四模型都已进入内部机制研究，但覆盖深度并不均衡。
`Gemma4` 现在已经不是黑盒，但它在多义词切换上仍偏弱。

### 7.5 最小因果回路仍只是起点

现在已经完成四模型的第一轮跨任务因果比较，但它仍然只是起点：

1. 目前仍主要围绕单一名词主线。
2. 还没有扩成多名词家族统一因果定律。
3. `GLM4` 的跨任务因果子集较弱，`Gemma4` 虽强，但还需要复核其是否普遍适用于更多概念家族。

### 7.6 工作方程还不是第一性原理

当前方程可以组织现象，但还不能精确预测全部语言行为。

### 7.7 预测能力仍停留在家族级

虽然 `stage522` 已经把家族预测准确率推到很高，但当前能稳定预测的主要还是：

1. 一个新名词更像哪个家族
2. 它会共享哪些家族骨干

还不能稳定精确预测：

1. 某个新名词的全部独有残差
2. 它在不同任务里的全部桥接细节
3. 它与更高层抽象结构的全部交互方式

### 7.8 层带动力学仍然是宽带，不是针尖

现在虽然已经看到明显层区，但还不能误解成“某种能力只在一个固定层里”。

更准确地说：

1. 当前看到的是宽带聚集，不是单层绝对专属。
2. 同一功能往往会跨若干相邻层传播。
3. 同一模型内部也常出现“早层残影 + 晚层主峰”的双带结构。
4. 所以层带图谱更像动力学地图，不像静态脑区图。

### 7.9 六类词性全景扫描仍主要限于两模型

当前六类词性的完整神经元级全景扫描，最稳的仍然主要来自：

1. `Qwen3`
2. `DeepSeek7B`

这意味着：

1. 我们已经能比较硬地描述六类词性的编码风格。
2. 但四模型对齐版六词类图谱还没有完成。
3. 尤其 `GLM4` 和 `Gemma4` 目前还缺完整六词类的同尺度神经元级扫描。

## 8. 下一阶段大任务

下一阶段不能再只补一个零散小功能，而要围绕四条主线推进。

### 8.1 多名词家族跨任务因果扩展

目标：

1. 把水果、动物、工具、组织继续扩成更多家族。
2. 把跨任务最小因果回路从苹果扩成多名词普遍规律。
3. 比较 `GLM4` 与 `Gemma4` 的跨任务因果回路是概念特异，还是风格特异。

### 8.2 绑定项突破

目标：

1. 把 `G_l` 从结构证据推进到强因果闭环。
2. 同时纳入对象骨干、属性通道、关系适配器和最小桥接子集。

### 8.3 名词机制四维坐标系

目标：

1. 统一量化共享底座强度。
2. 统一量化切换稀疏度。
3. 统一量化默认义偏置。
4. 统一量化控制杆显性度。

### 8.4 神经元级概念动力学

目标：

1. 把共享骨干、关系适配器、路由、绑定、低成本切换压进同一动态方程。
2. 找出神经元组合如何共同实现“海量概念、任意连边、高效率更新”。
3. 把“层带位置图谱”纳入动力学，回答哪些功能是早层路由、哪些是中层绑定、哪些是晚层收束。

### 8.5 名词编码预测定律

目标：

1. 把“全局骨干 + 家族骨干 + 名词独有残差”从工作规律推进成更严格的可预测模型。
2. 研究是否能利用已知家族成员，预测一个新名词的大部分编码骨架。
3. 研究剩余独有残差是否还能进一步分解成属性残差、关系残差和任务残差。

### 8.6 完整语言功能层区地图深化

目标：

1. 把动词、代词、连接词、固定搭配之外的更多语言功能继续纳入同一层区地图。
2. 研究同一能力在不同模型里为何出现“早层统一骨干型”“晚层写入型”“超晚层收束型”等不同实现。
3. 把层区地图从描述图谱推进到可预测图谱。

### 8.7 桥接项分型理论

目标：

1. 不再把桥接项只当成一个总模块，而是拆成属性桥接、关系桥接、语法桥接、联想桥接。
2. 比较不同桥接类型的稀疏度、层带位置、控制杆显性度。
3. 研究为什么某些模型更容易把桥接前置，而另一些模型更容易把桥接推到晚层。

### 8.8 四模型六词类对齐图谱

目标：

1. 把 `GLM4` 和 `Gemma4` 也补进六类词性的完整神经元级全景扫描。
2. 研究“六词类层带风格”到底是跨模型稳定规律，还是目前只在 `Qwen3 / DeepSeek7B` 上清楚。
3. 把词类层带风格和概念骨干、路由、绑定进一步并入同一统一动力学。

---

## 9. 当前阶段最严格的表述

如果只保留一句最严格的话，那就是：

**深度神经网络中的语言能力，当前最像一套由共享概念骨干、任务适配器、路由机制、绑定机制和低成本切换共同组成的分层控制系统；不同模型可能共享抽象分工，但不共享同一具体拓扑。**

距离第一性原理理论，还差三件事：

1. 把绑定项打穿。
2. 把控制变量继续压缩。
3. 把工作方程推进成可预测、可判伪、可跨模型复用的统一动力学。
