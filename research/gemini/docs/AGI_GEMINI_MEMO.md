

# 整体研究进展与路线图报告 (2026-02-28)
---
## 一、当前整体研究进展
本项目致力于实现基于第一性原理的人类水平智能系统（AGI），已彻底抛弃传统 BP 黑盒与单纯堆叠算力的路线。目前核心进展聚焦：
1. **理论根基确立**：建立基于微分几何、神经纤维丛拓扑（NFBT）和纯代数演化的智能引擎（Mother Engine）原型，并证明“极效三定律”。
2. **结构解剖与特征机制**：
   - 证实知识存放在高特异化、极度稀疏的专家神经元中（发现 DNN 惊人稳定具备 78% 的稀疏度和 97% 的正交性）。
   - 拆解 Attention 发现其工作在极低维度空间且行为单调，确立“满秩纤维存记忆，低秩流形走逻辑”的可分离架构。
3. **物理特征与自发涌现提取**：
   - 无全局梯度（BP）约束下，成功利用局部侧抑制结合 Hebbian 更新激发出高度稀疏编码底座（峰度攀升至 19.75）。
   - 捕获到了模型由“死记硬背”跨向“规律泛化”（Grokking）机制发生时的几何相变临界点（黎曼流形指标跃变）。

## 二、完整的路线图
整体研发路径分为以下战略梯队与微观探索阶段：

**核心战略层级规划**：
- **H1 阶段（已完成）**：理论奠基与小规模实证、微积分几何建模、可视化验证并网。
- **H2 阶段（当前攻坚）**：深度解剖现有大模型化石（如 GPT-2 分析），尝试局部学习机制突破，强攻信用分配（Credit Assignment）难题。
- **H3 阶段（中期目标）**：跨模态信号流统一，实现大规模语义自然涌现与百万 Token 连贯时序逻辑推演。
- **H4 阶段（远期目标）**：摆脱冯·诺依曼架构束缚，植入神经形态可控芯片（Neuromorphic Chip），打造低功耗、可干预的全尺寸 AGI 物理系统原型。

**“特征与编码机制”微观研究主线五步法**：
1. **特征涌现追踪**：通过从零训练刻画“特征从无到有、从分化到组合”的核心时间线。
2. **编码基本单位分析**：确认概念原子的存储尺度边界与组合规则。
3. **稀疏性与正交性物理定位**：探明 78% 稀疏与 97% 正交背后的必然结构。
4. **层级结构构建**：定位简单特征在宏观空间如何形成强抽象的高阶集合。
5. **大脑皮层多模态印证**：与真实神经科学数据交叉验算。

## 三、当前存在的问题和致命硬伤
1. **🔴 致命硬伤 - 信用分配危机 (Credit Assignment)**：在摒弃全局反向传播后，局部规则不知如何精准地把宏观误差分摊给底层突触。这导致模型根本无法在复杂特征上形成深度“专家化分工”（例如 SCRC 架构在 MNIST 任务上仅达 21% 准确率），卡死了 AGI 底层容量规模化的喉咙。
2. **🟠 严重瓶颈 - 语义连贯与符号接地断层**：纯代数引擎模型极易陷入局部几何势能洼地（如持续输出复用最高频的无意义死记字符），长期深度逻辑缺乏延展力。缺乏直接从感官像素流自发形成通用概念映射即“符号接地”的能力。
3. **证据链强度不均与架构漂移**：干预实验中对底层特征干预信号强而整层结构级证据偏弱，同时不同模型族对同样参数的敏感度差异明显，向真实复杂场景平移的确定性仍显不足。

## 四、接下来的核心工作 (Next Steps)
1. **P0 任务：攻关完整的分层预测编码 (Predictive Coding) 体系**
   - 直面信用分配危机。将假说大幅升级为 "竞争稀疏 + 预测编码" 双层耦合。顶层引入基于 Rao & Ballard 大一统框架的完整版分层独立预测器与误差回传机制，充当底层的无 BP 教师，争取短期内突破 85% MNIST 准确率指标。
2. **P0 任务：开发严谨特征涌现追踪工具与自动化管线**
   - 建立单源 JSON 产出的自动化提取归档管道来消除口径不一致；开发时序监视器，记录模型从随机出世起的每 100 步激活分布、特征演变及流形特征演化地图。
3. **P1 任务：大规模扫图实证与扩增多模态真实测试环境**
   - 开展多架构并行的剂量-响应维度分析（Layer / Top_k / Alpha）；引入多模态真实连续数据源集，彻底压低纯合成场景带来的能力评估偏置。

**二、完整研究路线图规划**

**三、面临的核心问题与致命硬伤**
-  **信用分配危机 (Credit Assignment)**：当前最大的死穴。切断了全局BP的上帝视角后，局部 Hebbian 等算法由于无清晰宏观误差传递，无法在极复杂层级间进行准确惩罚与责任分摊，致使实验模型无法完成深度专家群态的高级分化（例如 SCRC 在纯视觉图集上提取率依然偏低）。
-  **语义连贯与符号接地断层**：因为是纯代数引擎驱动，缺少常识物理规律的底座依托，导致模型轻易陷入局部势能洼地（如循环输出单调的无意义字符），难以维持连续深度逻辑，其符号接地（Symbol Grounding）仍不稳固。
-  **数据支撑单调**：当前过于依赖合成实验集与少数语言语料，脑科学实验观测验证的环节较为匮乏。

**四、接下来的实验部署与工作**

2. **[P0] 流形特征观测管线**：启动完全随机初态训练，每100步为单位高频切片，用可视化 3D 终端立体还原整个知识几何网络从无到结晶的相变全过程。
3. **[P1] 多模态并网与交叉验证**：打通图像和高级语义的数据隔离墙，开展跨模态联合响应敏感度验证。
