> 创建时间: 2026-04-26 08:48
> 核心原则: 先完成拼图, 不预设数学路线, 基于大量拼图逆向分析

## 一、为什么不能预设数学路线

**历史教训**: 从CCLI到CCLVIII, 每一次"理论突破"都在后续实验中被推翻:

| 预设理论 | 推翻原因 | 阶段 |
|---------|---------|------|
| 频分复用(CDMA) | 随机模型更正交, 训练降低了正交性 | CCXXXII |
| 语义引力 | PC1重叠与交互比不相关 | CCXXXIII |
| 亚加性压缩/低秩 | 交互项是高秩的 | CCXXXIV |
| 纤维丛结构 | 英文时态不是1维纤维 | CLXXXII-IV |
| 1维方向操作 | 1维方向注入无因果效应 | CLXXXV |
| 辛纠缠 | 辛形式跨层不守恒 | CCXXXVII |
| 曲率空间/纤维丛 | 曲率积分是传播假象 | CCXLIV-V |
| 深层非线性更强 | 激活范数归一化后反转 | CCXLVI |
| 句法-词汇双轨 | 只有question标准化, 其他句法特征不 | CCLIV |
| 解耦因果空间 | E/R指数增长是非线性假象 | CCLVII |
| 定向因果放大 | SA>1.0是采样不足假象 | CCLVII-VIII |
| 曲率梯度模型 | 曲率差异可能只是PCA对齐的推论 | CCLVIII |

**根本原因**: 大脑的3D神经网络结构极其复杂, 远超简单的几何/代数/拓扑框架。我们一直在用低维数学工具"解释"高维现象, 本质上是在做拟合, 而非发现。

**正确方法**: 收集大量可靠的实验拼图, 不急于统合, 等拼图足够多时再分析模式。

## 三、已被推翻的结论(反面拼图)

以下结论经过验证被推翻, 它们本身也是重要的拼图(告诉我们"不是什么"):

| 编号 | 被推翻的结论 | 推翻证据 | 教训 |
|------|------------|---------|------|
| X1 | "特征正交=频分复用" | 随机模型更正交 | 高维空间的正交是自然的, 不是学习策略 |
| X2 | "1维方向=因果1维" | 1维注入无效 | 统计维度≠因果维度 |
| X3 | "E/R指数增长=因果放大" | 线性区E/R≈1.0 | 大eps的非线性效应≠因果关系 |
| X4 | "SA>1.0=超越谱结构" | 精确SVD后SA<1.0 | 采样不足严重低估||J||_2 |
| X5 | "深层非线性更强" | 归一化后反转 | 激活范数增长的假象 |
| X6 | "深层解纠缠" | 累积曲率递增 | 深层在传播和放大纠缠, 不是解纠缠 |
| X7 | "浅层播种=纠缠生成" | CCXLV显示是传播距离效应 | 控制变量后才看到真相 |
| X8 | "余弦相似度可以测量纤维性" | 欧氏距离给出完全不同结论 | 高维空间中余弦不敏感 |
| X9 | "PC1重叠=信息重叠" | PC2完美分离 | 多分辨率编码 |
| X10 | "语义引力模型" | PC1|cos|与交互比不相关 | 重叠≠交互 |

## 五、新研究路线: 拼图优先, 逆向还原

### 核心原则
1. **不预设数学框架**: 放弃"几何/代数/拓扑"的先验假设
2. **拼图量>理论**: 每个实验产出一个硬事实, 不急于统合
3. **多层次验证**: 每个结论必须跨模型、有随机基线、大样本
4. **从3D大脑出发**: 思考什么样的3D结构能产生观察到的1D数据

### 语言能力解释力评估 (2026-04-26)

| 语言能力 | 现有解释力 | 可解释的部分 | 核心空白 |
|---------|-----------|------------|---------|
| 知识网络 | ~5% | 超分散编码(A3)、亚加性压缩(A4) | 概念编码位置、属性组织、抽象层级、网络拓扑 |
| 逻辑体系 | ~3% | 逻辑信号在浅层(C1) | 推理因果路径、条件链传播、前沿模式 |
| 语法体系 | ~8% | 语法因果逐层增长(C3) | 词性区分、句式模板、段落结构 |
| 多维生成控制 | ~3% | 无 | 风格控制路径、全局选择机制 |
| 系统效率 | ~8% | 分布式编码(B3) | 快速查找/修改的数学机制 |
| 词嵌入数学 | ~15% | 否定方向dim=1(A1)、跨语言一致(A9) | 线性结构完整性 |
| 全局唯一性 | ~5% | SA<1.0(B8) | 词选择的精确机制 |

**总解释力: 约5-7%** — 30+拼图全是统计性质, 对功能机制几乎空白

### Phase I': 从统计性质转向功能机制 (CCLIX-CCLXIV)

> 核心转变: 之前是"测统计量→猜机制", 现在是"选语言任务→追踪因果路径"

**CCLIX: 知识网络映射(Qwen3 bf16)**
- 选20个概念形成3条抽象链: 苹果→水果→食物→物体; 狗→动物→生物→物体; 车→交通工具→物体
- 追踪每个概念在残差流中的编码位置和维度
- 分析抽象层级间的编码关系(是否嵌套?是否正交?)
- 目标: 首次建立"知识拓扑→参数映射"的拼图

**CCLX: 推理因果路径(Qwen3 bf16)**
- 设计条件推理句对: "如果下雨, 地面会湿" vs "地面湿了" → 推断"可能下雨了"
- 追踪推理信号从条件到结论的层间传播
- 逻辑信号稀释(C1)后是否转化为其他形式?
- 目标: 首次追踪推理的因果路径

**CCLXI: 词性区分机制(Qwen3 bf16)**
- 同形异类词: run(名词vs动词)、light(形容词vs名词vs动词)、book(名词vs动词)
- 同一词在不同句法位置的残差流差异
- 词性信息是叠加在词义上还是分离编码?
- 目标: 理解词性的编码方式

**CCLXII: 全局选择机制(Qwen3 bf16)**
- 最终logit层的精确分析: Unembedding矩阵如何将残差映射到词表?
- Jacobian谱(SA<1.0)与词选择的关系
- 为什么4000+词中只选一个? softmax之前的logit分布形态
- 目标: 理解词选择的数学机制

**CCLXIII: 风格控制路径(Qwen3 bf16)**
- 同一内容不同风格: "太阳是恒星"(论文) vs "太阳就是个大星星"(聊天)
- 追踪风格信号的注入点(哪一层开始分化?)
- 风格是叠加在内容上还是独立通道?
- 目标: 理解多维生成控制的实现

**CCLXIV: Jacobian谱语义解码(Qwen3 bf16)**
- 原计划CCLIX: 逐层SVD + top奇异向量语义解码
- 特征方向在奇异谱中的精确位置
- 目标: 回答U2, U3(从功能视角)

### Phase II: SAE逆向 (CCLXV-CCLXX)

基于Phase I的硬事实, 训练和验证SAE:

**CCLXV: Delta-SAE训练**
- 对语法差分向量训练SAE(逐层累积的原子)
- 对语义差分向量训练SAE(MLP直接编码的原子)
- 目标: 获得原子特征

**CCLXVI: SAE特征因果验证**
- 沿SAE特征方向做因果干预
- 验证SAE特征是否比原始方向更有效
- 目标: 回答U9

**CCLXVII: SAE特征电路追踪**
- 追踪SAE特征从输入到输出的完整路径
- 哪些注意力头/MLP负责哪些SAE特征
- 目标: 理解信息路由

### Phase III: 逆向还原 — 从3D大脑结构出发 (CCLXXI+)

当拼图足够多时, 从大脑3D结构的角度分析:

1. **3D空间的约束**: 大脑皮层的6层结构、柱状组、连接拓扑
2. **快速读写的要求**: 什么结构允许O(1)检索和修改?
3. **特征提取的机制**: 3D空间中什么样的连接模式能提取复杂特征?
4. **Transformer作为3D结构的1D投影**: 40层×d_model维 = 3D皮层的"展开"

## 六、方法论警示

### 必须遵守的规则

1. **永远有随机基线**: 任何"结构发现"必须与随机初始化模型对比
2. **永远有eps扫描**: 任何结论必须在线性区(eps<1%)验证
3. **永远用欧氏距离**: 高维空间中余弦相似度不可靠
4. **永远用精确SVD**: 30-200个随机方向不足以估计||J||_2
5. **永远跨模型验证**: 单模型结论不可靠
6. **永远区分统计vs因果**: dim90=1≠因果1维

### 必须避免的陷阱

1. **不要因为"好看"就选数学框架**: 纤维丛/辛几何/曲率都"好看", 但都被推翻了
2. **不要过度外推**: 小样本→大结论是最常见的错误
3. **不要混淆相关和因果**: PCA对齐≠因果放大
4. **不要忽视量化效应**: 8bit/4bit改变了模型行为
5. **不要忽视范数效应**: ||h||增长10-180倍, 任何绝对量测量都会被扭曲
