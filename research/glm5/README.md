# GLM5 路线: 特征涌现与编码机制

## 核心问题

**神经网络如何从信号流中提取特征、形成编码？**

这是一切能力的基石。知识层次网络结构、高效读写、意识统一处理，都是建立在这个编码机制之上。

---

## 研究原则

### 1. 不预设理论正确

```
可能的错误预设:
✗ "稀疏编码是关键" → 可能只是现象，不是原因
✗ "纤维丛是数学基础" → 可能完全找错了方向
✗ "大脑和DNN用同样机制" → 可能只有表面相似
✗ "需要找到那个结构" → 可能根本不存在单一结构

正确态度:
✓ 观察DNN实际做了什么
✓ 收集足够多的"拼图块"
✓ 让结构自然浮现，而不是强加理论
```

### 2. 聚焦核心问题

```
问题链:
一切都是基石: 神经网络如何从信号流中提取特征、形成编码？
    ↓
在此之上形成: 特征如何形成层级结构？
    ↓
在此之上形成: 层级结构如何支持抽象与精确并存？
    ↓
在此之上形成: 不同模态如何统一编码？
    ↓
最终涌现: AGI能力
```

### 3. 方法论转变

```
旧方法: 理论假设 → 实验验证 → 修改理论
        问题: 理论假设可能根本就错了

新方法: 观察 → 记录 → 模式识别 → 假说 → 再观察
        核心: 先观察，后假说
```

## 研究计划

### Phase 1: 特征涌现追踪 (1-2个月)

```
目标: 观察"特征如何从无到有形成"

方法:
1. 从随机初始化开始训练
2. 每100步记录:
   - 激活分布变化
   - 稀疏度变化
   - 特征方向变化
   - 输出质量变化
3. 识别关键转变点

关键问题:
- 特征是什么时候出现的？
- 第一个"有意义"的特征是什么？
- 特征如何分化、组合？
- 涌现是否有阶段性？

预期获得:
├── 特征涌现的时间线
├── 涌现的临界条件
└── 涌现的顺序规律
```

### Phase 2: 编码基本单位分析 (2-3个月)

```
目标: 找到编码的"原子"

方法:
1. 分析单个神经元 vs 神经元群体
2. 分析最小可解释单元
3. 分析特征组合规则

关键问题:
- 一个特征对应多少神经元？
- 特征是连续还是离散？
- 特征之间如何组合？
- 有没有"基本特征集"？
```

### Phase 3: 稀疏性与正交性机制 (2-3个月)

```
目标: 理解"为什么是这些值"

关键问题:
- 稀疏度由什么决定？
- 为什么DNN稳定在78%？
- 正交性有什么功能？
- 能否强制改变这些值？
```
