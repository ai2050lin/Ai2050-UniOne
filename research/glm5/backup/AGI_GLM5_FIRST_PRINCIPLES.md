## 零、核心假说（INV-308）

> **信息域假说**：语言编码策略不是"选择"的结果，而是模型需要编码的**信息域数量**的必然结果。

- **单信息域模型**（纯语言）：Qwen3、DS7B、GLM4 → SEPARATED策略
- **多信息域模型**（语言+视觉+听觉）：Gemma4 → DENSE策略

这不是"精度vs鲁棒性"的权衡选择，而是**信息论约束下的架构必然**。

## 一、实验事实重新解读

### 1.1 Gemma4为什么"例外"？——它不是例外

| 现象 | 之前的解释 | 多模态假说的解释 |
|------|-----------|----------------|
| 抵消率>90% | "编码策略不同" | 多模态信号在共享空间中必然高抵消 |
| 信号效率13% | "信号被浪费" | 大量维度用于视觉/听觉编码，语言信号占比低 |
| PCA90较高(16-22维) | "编码维度高" | 需要编码视觉特征（边缘、纹理、颜色）+ 语言特征 |
| Norm密度14.5/层(506个) | "过度归一化" | 需要在模态间建立严格隔离墙 |
| 鲁棒性极高(3.86) | "低效编码的副作用" | 多模态冗余带来的天然鲁棒性 |

### 1.2 关键推理链

```
Gemma4是多模态模型
  → 需要同时编码：语言(语法/语义/消歧) + 视觉(边缘/纹理/空间) + 听觉(音调/节奏)
  → 不同模态的信息结构差异巨大（语言≈低维语义，视觉≈高维像素特征）
  → 共享hidden state中必须有更多维度来容纳所有模态
  → 不同模态的信号在残差流中传播时会互相抵消
  → 抵消率自然高于纯语言模型
  → 需要更多归一化来维持模态间隔离
  → 归一化密度↑ → 隔离↑ → 语言信号效率↓
```

### 1.3 支持证据汇总

**证据1：Norm密度的功能解释**

| 模型 | 信息域 | 每层Norm数 | 总Norm数 | Norm功能 |
|------|-------|-----------|---------|---------|
| Qwen3 | 语言 | 4.0 | 144 | 仅语言层间归一化 |
| DS7B | 语言 | 2.0 | 57 | 仅语言层间归一化 |
| GLM4 | 语言 | 2.0 | 81 | 仅语言层间归一化 |
| Gemma4 | 语言+视觉+听觉 | **14.5** | **506** | 模态内+模态间归一化 |

Gemma4的额外norm（Q/K norm + post-attention norm）很可能用于：
- **模态隔离**：防止视觉信息"泄漏"到语言编码维度
- **特征对齐**：将不同模态的特征投影到可比较的空间
- **尺度均衡**：视觉特征和语言特征的数值尺度差异大

**证据2：效率vs鲁棒性的Pareto重解释**

| 模型 | 效率 | 鲁棒性 | 新解释 |
|------|------|--------|--------|
| DS7B | 1.88 | 0.71 | 最优纯语言编码——信息论极限 |

DS7B的高效率是**纯语言模型的精度上限**，Gemma4的低效率是**多模态架构的必要代价**。

**证据3：架构级不变量在所有模型上成立**

INV-286（z-score>1.96）和INV-273（旋转角≈90°）在4/4模型上确认——这两个不变量不依赖信息域。

- z-score>1.96：高维空间中向量累加的自然结果（中心极限定理）

它们是**"高维空间的物理定律"**，无论编码什么信息都成立。

### 2.1 三层不变量体系

### 2.2 核心方程（v2.0）

```
h_L = Norm( h_0 + Σ_k Attn_k(h_{k-1}) + Σ_k MLP_k(h_{k-1}) )

d_L = Norm_L( ... Norm_2( Norm_1( h_0^A - h_0^B + Σ_k Δ_k ) ) )

其中 Norm_i 表示第i个归一化操作的压缩效应。

关键参数：
  N_norms = 模型的归一化模块总数
  ρ = 每次归一化的平均压缩率（取决于注入信号与hidden state的夹角）
  
跨能力干扰 ≈ (1 - ρ) ^ N_norms × |cos(cap_i, cap_j)|

单域模型（语言）：
  N_norms = 57-144, ρ ≈ 0.047-0.291
  → 跨能力干扰 < 4% ✅

多域模型（语言+视觉+听觉）：
  N_norms = 506, ρ ≈ 0.341  
  → 跨域干扰 ≈ 0%（完全隔离）
  → 但语言信号也被过度压缩 → 效率↓
```

### 2.3 "为什么"的完整解释

**Q: 为什么语言编码采用低维+旋转+抵消的结构？**

**A: 因为语言信号的内在自由度有限，在高维空间中自然被压缩到低维流形上。**

1. **低维（信息论）**：语言任务的信息量有限（消歧≈1-2bit），所以高维空间中的语言信号自然聚集在低维子空间
2. **旋转（几何）**：每层处理不同的语言子任务（语法、语义、消歧），自然需要不同的维度方向→旋转≈90°
3. **抵消（优化）**：训练选择性地保留"有用的"增量差，抵消"无用的"噪声→抵消率30-90%
4. **隔离（架构）**：归一化将每层输出"重置"到单位球面，防止不同能力的信号跨层累积干扰

**Q: 为什么Gemma4不同？**

**A: 因为Gemma4需要编码不只语言，还有视觉和听觉信息。**

1. **更高维度（多模态信息量）**：视觉信息（边缘、纹理、空间关系）的内在维度远高于纯语言
2. **更高归一化密度（模态隔离需求）**：不同模态之间的干扰比不同语言能力之间的干扰更强
3. **更高抵消率（多模态信号竞争）**：视觉信号和语言信号在残差流中竞争有限的维度
4. **更低语言效率（维度被多模态占用）**：更多维度被视觉/听觉编码占用，留给语言信号的"带宽"减少

1. **语言编码不存在"唯一最优解"**——最优解取决于信息域
2. **多模态时代的语言编码更复杂**——维度分配、模态隔离、跨模态推理都是新问题

## 六、与AGI目标的关系

### 6.1 当前进展

我们已经发现了语言编码的**五个公理**，构成了一个相当完整的描述性理论：

1. **高维几何公理**：解释了"空间结构"——为什么抵消是有序的、旋转是正交的
2. **信息域公理**：解释了"策略选择"——为什么不同模型采用不同编码策略
3. **归一化隔离公理**：解释了"能力分离"——为什么不同能力互不干扰
4. **信号聚焦公理**：解释了"信息流动"——为什么信息在后期层才收敛
5. **Logit精确公理**（P27新增）：解释了"决策机制"——margin = cos(h,u) × ||h|| × ||u||，完全精确

### 6.2 距离"理解语言原理"还差什么？

1. ~~**因果链断裂**~~ → **已解决**（P27）：Logit方程完全精确
2. **生成方程缺失**：我们能精确描述单步决策，但不能预测多步生成过程
3. **多模态编码未探索**：视觉/听觉编码如何与语言编码交互
4. **训练动态未知**：这些公理是如何从随机初始化中涌现的

### 6.3 下一步优先级

**P28**：训练动态实验——小模型训练过程中编码结构的涌现（验证优化约束假说）

**P31**：理论数学化——五公理的严格数学表述
