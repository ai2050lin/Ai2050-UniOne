# AGI_GLM5_LANGUAGE

## 0. 文档说明

这份文档记录GLM5线路的DNN语言编码机制研究。

研究方法：从参数级和神经元级的原始数据出发，还原语言在深度神经网络中的编码机制。
研究目标：逼近"语言背后的编码机制，怎样在残差流中组织、偏转、放大，并最终生成语义、语法、风格和逻辑。"

数据来源：
- Phase 1: Embedding SVD、残差流几何、FFN变换类型分类
- Phase 2: 残差流方向分析、有效秩修复、注意力差异修复、方向累积
- Phase 3: 螺旋轨迹分类、中间层深度分析、注意力头功能、跨模型验证
- Phase 4: 子空间投影、token偏离、逐层判别力、因果干预
- Phase 5: 真正的因果干预实验、FFN旋转器精确分析

使用模型：GPT-2 (768d/12层/12头), Qwen2.5-0.5B (896d/24层/14头), Qwen2.5-1.5B (1536d/28层/12头)
硬件：RTX 3080 Laptop 8GB

---

## 1. 研究原则

### 1.1 总原则

1. 先看原始数据，再看结构角色，再看因果链，最后才审理论资格
2. 不把阶段性工作公式当成终局理论，只当成工作性原理和第一性原理候选
3. 严格区分"大脑编码"和"DNN编码"——DNN提取了大脑中的数学结构，但架构不同
4. 从"静态参数分析"转向"动态计算分析"——关注计算产生了什么，而非参数编码了什么

### 1.2 核心差异：DNN vs 大脑

DNN将词嵌入和计算分开（Embedding层 → Transformer层），大脑中这两者是一体的。
因此，不能简单地把DNN的参数编码直接映射到大脑编码，而应关注DNN中的**计算动态**。

---

## 2. 核心目标

**从参数级和计算级还原DNN中的语言编码机制。**

当前最接近的工作性主链：

**残差流螺旋运动 → 语义流形收敛 → 子空间维度分离**

---

## 3. 当前最新结论

### 3.1 最稳的结构结论（跨模型一致，多Phase验证）

**DNN中的语言编码是一个三层系统（经Phase 5因果验证修正）：**

1. **基础层（涌现螺旋）**：残差流在低维子空间（有效秩2-6）中做螺旋运动，每层偏转约8-15°。这是架构级的不变量（Phase 2-3）。但Phase 5C揭示：螺旋不是FFN的预设行为（FFN线性旋转仅1.7°），而是注意力+非线性+残差连接的**涌现现象**。跨层FFN方向近乎随机（相关-0.074）。

2. **分离层（子空间投影的统计关联，非因果机制）**：不同语言维度的差异集中在PCA的Top-1方向（>94%）。但Phase 5A因果干预显示：零化该差异方向不能使A趋近B（16对中9对恶化）。**子空间投影是统计关联，不是因果编码机制**。

3. **放大层（范数增长）**：维度差异通过残差流范数的指数级增长被放大。GPT-2增长7.7x，Qwen-0.5B增长19.3x，Qwen-1.5B增长27.7x。语义维度的差异放大最强，逻辑维度最弱。

4. **加工层（中间层精加工）**：Phase 5C发现实际残差流偏转呈"U型"分布——浅层/深层偏转40-83°，中间层(L3-9)仅1-4°。中间层是精细信息加工的核心区域。

## 4. 结论的原始数据支撑

### 4.1 残差流做螺旋运动（Phase 2, 3验证）

**原始数据**：

```
GPT-2 (12层):
  每层平均偏转: 12.4~13.4°
  累积偏转总量: 163~169°
  所有8个测试句子差异 < 2°

Qwen2.5-0.5B (24层):
  每层平均偏转: 11.9~21.8°
  累积偏转总量: 275~295°

Qwen2.5-1.5B (28层):
  每层平均偏转: 7.8±0.1°
  累积偏转总量: 211°
```

**关键验证**：
- Phase 3扩展到16个测试句子（每维度4个变体），同维度内轨迹相关>0.993
- 跨维度轨迹相关同样>0.993，说明螺旋轨迹本身不携带维度信息
- 每层偏转角与模型深度成反比（15°→13°→8°），是一个缩放定律

**结论**：

**残差流的螺旋运动是架构级不变量，由模型权重矩阵的谱性质决定，不由输入内容决定。FFN 97%+是旋转器（Phase 1），每层产生微小方向偏转，多层累积形成螺旋轨迹。**

### 4.2 维度信息编码在低维子空间中（Phase 4核心发现）

**原始数据**：

```
子空间投影分析（GPT-2, 最后一层）:
  所有维度对的PCA-Top1占比: 94.7~97.5%
  所有维度对的PCA-Top5占比: 98.0~99.0%
  PCA-Top50占比: 0% (说明差异集中在前几个方向)
  正交于全局方向的差异: 87~100%
  逐token差异标准差: 29~120

Qwen2.5-0.5B:
  PCA-Top1占比: 45.3~92.7%
  PCA-Top5占比: 98.5~99.8%
  正交于全局方向的差异: 88~100%
```

**因果干预验证**：

```
残差流差异能量剖面（GPT-2）:
  syntax:  Layer 0-1: 3~13 → Layer 2-11: 105~129 (中层爆发)
  logic:   Layer 0-1: 2~10 → Layer 2-11: 85~108
  style:   Layer 0-1: 3~14 → Layer 2-11: 137~186
  semantic: Layer 0-1: 7~27 → Layer 2-11: 217~280 (最大)
```

**结论**：

**语法/逻辑/风格/语义的差异被压缩在1-5维的低维子空间中。差异方向与全局残差流方向高度重合（正交比>87%），说明维度差异不是在独立的"通道"中编码，而是在残差流主方向上的投影差异。**

### 4.3 语义是最强维度，逻辑是最弱维度（Phase 3, 4验证）

**原始数据**：

```
维度判别力（GPT-2, 逐层余弦相似度差异）:
  Layer 0-1:  syntax=0.062, logic=0.070, style=0.049, semantic=0.042
  Layer 9-11: syntax=0.027, logic=0.034, style=0.029, semantic=0.017

分歧值（跨句子对）:
  GPT-2:   syntax=0.267, logic=0.014, style=0.288, semantic=0.374
  Qwen:    syntax=0.640, logic=0.024, style=0.783, semantic=0.829

因果干预 logit差异:
  GPT-2:   syntax=59, logic=12, style=60, semantic=108
  Qwen:    syntax=2.1, logic=3.4, style=3.3, semantic=6.7

残差流能量增长:
  GPT-2:   syntax=129, logic=108, style=186, semantic=280
```

**结论**：

**语义在残差流中编码最强（分歧最大、logit差异最大、能量增长最快），逻辑编码最弱（分歧<0.03，logit差异最小）。语法和风格居中。这暗示DNN对逻辑推理的编码机制与语义理解有本质不同——逻辑可能依赖跨步骤的计算过程而非单步表征。**

### 4.4 Token偏离通用螺旋的角度约45-70°（Phase 4）

**原始数据**：

```
GPT-2: 平均偏离 45~49° (各维度差异不大)
Qwen:  平均偏离 64~70° (略高于GPT-2)
```

**结论**：

**个别token偏离通用螺旋约45-70°，但偏离角度在各维度间差异不大。这意味着token级的偏离虽然存在，但不是维度信息的主要编码方式。维度信息的编码主要在残差流差异的子空间结构中（4.2节）。**

### 4.5 归一化策略决定残差流几何演化（Phase 3跨模型对比）

**原始数据**：

```
有效秩变化趋势（最后层 - 第一层）:
  GPT-2 (LayerNorm):      +0.47 (16/16句子正增长)
  Qwen-0.5B (RMSNorm):    -1.00 (14/16负增长)
  Qwen-1.5B (RMSNorm):    -2.64 (16/16负增长)
```

**结论**：

**LayerNorm使有效秩递增（方向持续分散），RMSNorm使有效秩递减（方向持续收敛）。这是归一化策略的直接后果：LayerNorm重新归一化方向，RMSNorm只缩放范数。"先分散后收敛"只适用于RMSNorm架构。**

## 5. 结论推导过程

### 第一步：确认残差流的基础几何结构

做法：
- Phase 1: Embedding SVD、残差流范数逐层追踪、FFN变换类型分类
- Phase 2: 修复有效秩计算、逐层方向偏转角度测量

结论：残差流在有效秩2-6的低维子空间中运动，每层偏转约13°，累积超过180°

### 第二步：验证螺旋运动是否携带维度信息

做法：
- Phase 3: 扩展到16个测试句子，4个维度各4个变体，三模型验证

结论：螺旋轨迹不携带维度信息（同维度内相关>0.993，跨维度相关>0.993）

### 第三步：定位维度信息的编码位置

做法：
- Phase 4: 子空间投影分析、token偏离分析、逐层判别力、因果干预

结论：维度差异集中在PCA的Top-1~5方向（解释>95%方差），与全局残差流方向高度重合

### 第四步：验证维度间差异的量级和层级

做法：
- Phase 2-4: 跨维度角度差异、分歧层分析、logit差异、能量剖面

结论：语义最强，逻辑最弱，分歧在中层达到峰值

---

## 6. 当前最严格的总判断

**DNN中的语言编码系统可以概括为"涌现螺旋假说"：**

1. **基底**：残差流在低维子空间中做螺旋运动（架构级不变量）
2. **涌现**：螺旋不是FFN的预设行为，而是注意力+非线性+残差连接的涌现现象
3. **编码**：维度差异的统计投影集中在PCA-Top1方向，但这不是因果机制（Phase 5A否定）
4. **加工**：中间层(L3-9)是精细信息加工区域（偏转1-4°），浅层/深层是粗放变换区（偏转40-83°）
5. **放大**：范数增长放大维度差异，语义>风格≈语法>>逻辑

**与Phase 4"螺旋子空间假说"的关键变化**：
- "子空间投影是因果编码机制" → **被否定**（Phase 5A）
- "FFN是旋转器" → **部分修正**：FFN线性旋转仅1.7°，非线性才是主要旋转源
- 新增"中间层精加工"概念（Phase 5C的U型偏转分布）

1. **~~FFN谱分析未能完成~~** (Phase 5C已修复) — FFN线性旋转仅1.7°/层，有效秩511，跨层方向不相关

4. **逻辑维度的编码机制仍不明确**：逻辑推理在因果干预中受害最严重（3/4对大幅恶化），说明逻辑对方向极度敏感但编码方式特殊
5. **非线性激活对旋转的贡献未量化**：FFN线性旋转1.7° vs 实际15°，但GELU具体贡献了多少仍需分离
6. **只有GPT-2和Qwen两个架构**：缺少Llama、Mistral等其他架构的验证

## 9. 当前结论

**一句话总结当前状态：**

DNN中的语言编码不是一个高维自由空间中的分布式表征，而是一个低维子空间（有效秩2-6）中的有结构螺旋运动。不同语言维度通过在这个螺旋上的微小投影差异来编码，这些差异通过范数的指数增长被放大为可用的语义信息。螺旋运动是架构级的不变量（所有输入共享），维度差异是内容相关的微小扰动。

**当前的判断级别**：

"涌现螺旋假说"是经过因果验证修正的候选结构。Phase 5A的因果干预否定了"子空间投影=因果编码"的简单假说，Phase 5C揭示了螺旋的涌现本质。当前理论的主要不确定点：(a) 维度信息的真正因果编码机制（可能是动态轨迹），(b) 非线性激活vs注意力的旋转贡献分离，(c) 逻辑维度的特殊编码方式。
