# AGI_GLM5_LANGUAGE

## 0. 文档定位

这份文档不是实验流水账，也不是某一次阶段汇报。
它的目标只有一个：

**系统化讲清楚当前对"深度神经网络语言编码机制"的研究进展，包括原则、目标、测试体系、核心拼图、当前结论、问题硬伤和下一阶段计划。**

当前最重要的背景判断是：

1. 深度神经网络已经提取了某种高度压缩的语言数学结构。
2. 这套结构既承载知识网络，也承载语法、风格、逻辑和概念关联。
3. 它很可能不是"词典式存储"，而是"分层控制系统"。
4. 现在的工作不是先发明一套漂亮理论，而是先通过大量拼图逼近真实结构。

---

## 1. 研究原则

### 1.1 先拼图，后理论

当前主线不是先写公理，而是先还原机制。
严格顺序应当是：

1. 先积累可复核的实验拼图。
2. 再压缩出稳定结构。
3. 再做因果打击和跨模型复核。
4. 最后才讨论是否有资格上升为第一性原理理论。

最核心的一句话是：

**先机制，后定理；先还原，后抽象。**

### 1.2 先因果，后修辞

一个结论想被保留，尽量要满足：

1. 能在原始激活或原始行为里直接看见。
2. 能被消融或干预打中，不只是相关。
3. 能跨样本复现。
4. 能跨模型复现。
5. 能被更强实验推翻或修正。

### 1.3 先统一变量，后局部故事

当前最危险的错误是：

1. 把局部亮点误当成全局机制。
2. 把工程噪声误当成理论信号。
3. 把单个模型风格误当成普遍定律。

因此当前研究始终优先寻找统一控制变量，而不是堆更多孤立个案。

### 1.4 GLM5线路的额外原则

1. 严格区分"大脑编码"和 `DNN（深度神经网络）` 编码—— `DNN` 提取了大脑中的数学结构，但架构不同。
2. 从"静态参数分析"转向"动态计算分析"——关注计算产生了什么，而非参数编码了什么。
3. 早期 `Phase 1-5` 的残差流螺旋研究提供了几何基底视角，与后期 `stage423-531` 的神经元级因果研究互为补充。

## 2. 核心目标

当前总目标不是证明模型"会说话"，而是回答下面这些更硬的问题：

1. 语言信息在网络内部如何分层组织。
2. 名词、属性、关系、代词、功能词、词元补全是否共享同一套高层控制结构。
3. `attention head（注意力头）`、`MLP neuron（多层感知机神经元）`、`residual stream（残差流）` 分别承担什么角色。
4. 这些能力是稀疏小回路实现，还是分布式场实现。
5. 不同模型是否共享同一抽象机制，只是拓扑不同。
6. 能否把这些拼图压成统一状态方程。
7. 这套编码机制的数学结构是什么，是否超出现有数学体系。

## 4. 当前统一工作方程

截至目前，有两层方程。

### 4.1 几何基底方程（Phase 1-5，残差流级）

残差流在低维子空间（有效秩2-6）中做螺旋运动，每层偏转约8-15度。这个几何基底是架构级不变量。

```
h_{l+1} = R_l(h_l) + f_l(h_l)
```

其中 `R_l` 是螺旋旋转（架构级不变量），`f_l` 是内容相关的信息写入。

关键定量：

| 性质 | GPT-2 | Qwen-0.5B | Qwen-1.5B |
|------|-------|-----------|-----------|
| 每层偏转角 | 15.0±0.2° | 12.6±0.3° | 7.8±0.1° |
| 总偏转角 | 165° | 290° | 212° |
| 范数增长 | 7.7x | 19.3x | 27.7x |
| 有效秩趋势 | +0.47（上升） | -1.00（下降） | -2.64（下降） |

### 4.2 功能分解方程（stage423-531，神经元级）

截至目前，最稳的工作方程是：

`h_t^{l+1} = h_t^l + C_l(form_t, context_t) + R_l(ref_t, context_t) + B_l(lemma_t) + S_l(lemma_t, context_t) + A_l(attr_t, context_t) + G_l(B_l, A_l, C_l, R_l) + O_l(h_t^l)`

这不是最终定理，而是当前最稳的工作模型。

各项当前含义如下：

- `C_l`
  词形连续性控制项。负责把局部字形或前缀推到高概率后续词元。

- `R_l`
  路由项。负责代词、功能词、句法角色和跨位置依赖的路由。

- `B_l`
  概念骨干项。负责共享概念底座、家族位置和公共对象承载。

- `S_l`
  切换项。负责多义词在共享底座上的低成本切换。

- `A_l`
  属性项。负责颜色、味道、大小、功能、角色等修饰方向。

- `G_l`
  绑定项。负责把对象、属性、关系和当前实例拼起来。

- `O_l`
  读出项。负责把内部状态压成最终输出。

从更高层抽象看，当前最稳的分工仍是：

1. `attention head` 更像负责路由、搬运、配对、条件选择。
2. `MLP neuron` 更像负责局部写入、目标放大、落点锁定。
3. `residual stream` 更像负责共享工作记忆和层间传播。

### 4.3 统一层带动力学（stage526-531）

`stage526` 到 `stage531` 把层带位置图谱和最小因果回路压到了一起。

当前最稳的统一表述是：

`h_t^{l+1} = h_t^l + E_l(route_t, function_t) + M_l(bind_t, role_t) + L_l(content_t, readout_t)`

其中：

- `E_l`
  早层路由项。更偏代词、引用建立、局部功能词控制、前置骨干写入。

- `M_l`
  中层绑定项。更偏关系桥接、语法角色整合、对象与接口的耦合。

- `L_l`
  晚层收束项。更偏名词、属性、固定搭配、最终答案读出。

## 5. 核心拼图

### 5.1 残差流螺旋基底（Phase 1-5，GLM5线路独特贡献）

**最稳的结构结论（跨模型一致，多Phase验证）：**

1. **基础层（涌现螺旋）**：残差流在低维子空间（有效秩2-6）中做螺旋运动，每层偏转约8-15°。这是架构级的不变量。但 `Phase 5C` 揭示：螺旋不是 `FFN` 的预设行为（ `FFN` 线性旋转仅1.7°），而是注意力+非线性+残差连接的**涌现现象**。跨层 `FFN` 方向近乎随机（相关-0.074）。

2. **分离层（统计关联，非因果机制）**：不同语言维度的差异集中在 `PCA` 的 `Top-1` 方向（>94%）。但 `Phase 5A` 因果干预显示：零化该差异方向不能使A趋近B（16对中9对恶化）。**子空间投影是统计关联，不是因果编码机制。**

3. **放大层（范数增长）**：维度差异通过残差流范数的指数级增长被放大。语义维度的差异放大最强，逻辑维度最弱。

4. **加工层（中间层精加工）**：`Phase 5C` 发现实际残差流偏转呈"U型"分布——浅层/深层偏转40-83°，中间层仅1-4°。中间层是精细信息加工的核心区域。

### 5.2 代词与功能词路由

当前最稳的功能词主线仍然是代词。

结论：

1. `DeepSeek7B` 的代词机制已经露出早层头组骨架。
2. 机制更像"先路由，后整合"。
3. 这是 `R_l` 最硬的一条证据线。

### 5.3 多义词切换

多义词主线已经不是苹果个案。

当前四模型分型如下：

- `Qwen3`：低重合弱因果型
- `DeepSeek7B`：低重合弱因果型
- `GLM4`：结构分裂型
- `Gemma4`：默认义偏置型弱切换

这说明：

1. 普通上下文变化和多义词跨义变化不是同一种机制。
2. 共享底座 + 切换结构是强工作假说。
3. 但不同模型露出的"切换控制杆"显性度不同。

### 5.4 苹果个案

苹果仍然是最深的局部个案。

当前更稳的图景是：

1. 苹果不是"水果苹果一套编码 + 品牌苹果一套编码"。
2. 更像"共享底座 + 切换结构 + 任务内修饰"。
3. `Qwen3` 更偏头骨架型，`DeepSeek7B` 更偏锚点增强型。

### 5.5 属性绑定

当前最稳的是结构定律，不是最小强因果闭环。

结论：

1. "对象骨干 + 属性通道 + 桥接区"已经从苹果个案扩成一般规律。
2. 但 `G_l` 仍然是最薄弱环节。
3. `size（大小）` 家族比 `color（颜色）` 和 `taste（味道）` 更早露出可压缩因果回路。
4. 四模型桥接因果搜索表明，部分模型已经出现可压缩的小型桥接因果子集：
   - `Qwen3`：桥接子集较弱，但方向正确
   - `DeepSeek7B`：已有小幅稳定桥接因果
   - `GLM4`：桥接下降明显，但控制偏移也偏大

3. 名词层带、属性层带、桥接层带通常相近，但不是完全重合。

当前四模型图谱：

- `Qwen3`
  - 名词/属性/桥接峰值层：`L34-L35`
  - 解释：明显偏晚层聚集，属于"晚层写入型"

- `DeepSeek7B`
  - 名词/属性/桥接峰值层：`L27-L26`，并伴随早层 `L3`
  - 解释：以晚层大峰为主，同时保留早层路由残影

- `GLM4`
  - 名词/属性/桥接峰值层：`L39-L38`
  - 解释：极强的末层聚集，属于"超晚层收束型"

- `Gemma4`
  - 名词/属性/桥接峰值层：`L4-L1-L0`
  - 解释：明显偏早层聚集，属于"早层统一骨干型"

### 5.10 大量名词全景层级扫描

`stage522` 和 `stage523` 说明：

1. 大量名词的编码可以分解成三层结构：全局共享名词骨干 + 家族共享骨干 + 名词独有残差。
2. 同家族名词之间的共享显著高于跨家族共享。
3. 这种结构已经足以支持家族级预测。
4. 家族预测准确率四模型都达到 `1.0`。

当前最稳的工作规律是：

**名词编码 = 全局名词骨干 + 家族骨干 + 名词独有残差**

### 5.11 完整语言功能层区地图

`stage524` 把已有的词类层分布、中文模式图谱和名词-属性层带图谱合到了一起。

当前关键图景：

- `Qwen3`：整体更像"中层组织，晚层收束"
- `DeepSeek7B`：整体更像"早层路由，中晚层整合"
- `GLM4`：更像"超晚层收束型"
- `Gemma4`：更像"早层统一骨干型"

### 5.12 桥接因果扩展

`stage525` 把桥接因果从"名词-属性"扩到了三条更广的主线：

1. `noun_relation（名词-关系）`
2. `noun_syntax_role（名词-语法角色）`
3. `noun_association_network（名词-联想网络）`

说明概念骨干与任务接口之间的连接本身也可以被压成小型因果子集。

### 5.13 六类词性全景扫描

`stage527` 到 `stage531` 把六类词性拉进了同一张全景图里。

最通俗的总结是：

1. 名词更像对象骨干带。
2. 形容词更像修饰双峰带。
3. 动词更像动作与过程带。
4. 副词更像调制带。
5. 代词和介词更像功能路由带。

严格来说：

1. 词类之间在 `top256` 神经元集合上的交并比大多接近 `0`，说明在"最尖锐控制杆"层面它们高度分离。
2. 但层带质心和宽带区域仍然会相邻甚至部分重叠。
3. 更合理的说法是：词类在最尖锐控制杆上高度分离，词类在宽带功能区上仍共享更高层组织。

### 5.14 四模型统一语言动力学

`stage531` 把全部拼图压成统一视图。

统一公式：`h_t^{l+1} = h_t^l + E_l(route_t, function_t) + M_l(bind_t, role_t) + L_l(content_t, readout_t)`

核心结论：更接近事实的结构不是均匀网络，而是**"宽带功能区 + 尖锐控制杆 + 小型桥接回路"**。

## 6. 当前最稳结论

截至目前，最稳的结论可以压成下面十四句：

1. 语言编码不像静态词典，更像分层控制系统。
2. `attention head`、`MLP neuron`、`residual stream` 是分工结构，不是同一种部件。
3. 残差流在低维子空间中做螺旋运动，这是架构级不变量（几何基底）。
4. 但子空间投影是统计关联，不是因果编码机制——维度信息的真正因果载体仍需在动态中寻找。
5. 功能词和代词已经露出路由骨架。
6. 多义词已经露出共享底座 + 切换结构。
7. 属性绑定已经露出对象骨干 + 属性通道 + 桥接区的结构定律。
8. 中文词元补全、功能词、固定搭配可以被压入同一图谱。
9. 同一个名词跨知识、语法、属性、联想任务确实复用同一部分骨干。
10. 跨任务最小因果子集已经开始在四模型中显形，说明概念骨干不是纯相关幻觉。
11. 有效神经元通常具有明显层带偏置，语言结构更像"宽带功能区"而不是均匀平铺。
12. 大量名词的编码已经露出"全局骨干 + 家族骨干 + 名词独有残差"的层级结构。
13. 把层带位置图谱与最小因果回路压到一起之后，当前最稳的统一动力学是：早层更偏路由，中层更偏绑定，晚层更偏收束读出。

## 7. 问题、硬伤与瓶颈

当前理论还远没有闭环，硬伤很明确。

### 7.1 绑定项已被重新定性：从"最弱"到"场控制杆"

`stage535-537` 已经把绑定项从"最弱环节"重新定性为"分布式场控制杆"：
1. 绑定瓶颈层存在（效率比>1.1），但不存在绑定瓶颈神经元（Jaccard≈0）。
2. 消融整层无法显著削弱绑定——因为绑定信息分布在数千个神经元上。
3. **新方向**：G_l不应写成 `g(h_i)`（单神经元），而应写成 `f(mean(h), std(h), entropy(h))`（统计量）。
4. 遗留问题：如何实验验证"场控制杆"假说？需要设计基于统计量变化的因果实验。

### 7.2 阈值依赖仍然存在

很多复用数、差异数、交并比仍依赖 `top-k` 定义，不能误当自然常数。

### 7.3 几何基底与因果机制的鸿沟

`Phase 1-5` 发现了残差流螺旋是架构级不变量，但也证明了子空间投影不是因果编码机制。
这意味着：
1. 维度信息的真正因果载体仍不明确。
2. 螺旋几何只是"舞台"，真正在舞台上表演的"演员"还需要被识别。
3. 几何基底研究和神经元级因果研究需要进一步融合。

### 7.4 因果效应量偏小

跨任务因果子集的效用值大多在 0.001-0.025 之间，方向正确但效应量偏小。
这可能说明：
1. 当前方法还不够精确
2. 真正的控制杆可能更分布式
3. 或者需要更大规模的消融

### 7.5 四模型并非同一种可观测深度

虽然四模型都已进入内部机制研究，但覆盖深度并不均衡。
`Gemma4` 现在已经不是黑盒，但它在多义词切换上仍偏弱。

### 7.6 最小因果回路已扩展到六类名词家族

`stage532-534` 已完成六类名词家族（水果/动物/工具/组织/天体/抽象）的统一因果搜索。
遗留问题：
1. 编码距离用单token计算，应该用完整句式中的编码。
2. 语法任务被跳过（N/A），实际只用3种任务。
3. GLM4和Gemma4尚未纳入。

### 7.7 工作方程还不是第一性原理

当前方程可以组织现象，但还不能精确预测全部语言行为。
方程中的各项只是变量名，没有闭式数学表达。

### 7.8 预测能力仍停留在家族级

当前能稳定预测的主要是一个新名词更像哪个家族、会共享哪些家族骨干。
还不能稳定精确预测某个新名词的全部独有残差、全部桥接细节。

### 7.9 层带动力学仍然是宽带，不是针尖

当前看到的是宽带聚集，不是单层绝对专属。
层带图谱更像动力学地图，不像静态脑区图。

### 7.10 六类词性全景扫描仍主要限于两模型

四模型对齐版六词类图谱还没有完全完成。
`GLM4` 和 `Gemma4` 的六词类扫描是缩减版。

### 7.10b "维度坍缩"被推翻（stage577）

之前的"维度坍缩在15-17%层"是基于单token的有效秩计算。但单向量的有效秩永远是1，这是数学限制而非物理坍缩。
需要用完整句式中所有token的hidden state矩阵来计算有效秩，重新定义和验证。

### 7.11 bank/apple二元对立被打破（stage573）

之前的"bank走attention路径、apple走MLP路径"是误导性简化。实际是：**attention是默认消歧机制(80%)**，MLP是少数特例(plant/nail)，apple实际上是混合型。
这要求重新审视所有基于bank/apple对比得出的结论。

### 7.12 家族预测精度差异巨大（stage581）
