# AGI_GLM5_LANGUAGE

## 0. 文档定位

这份文档是"深度神经网络语言编码机制"研究的**系统性总览**。

核心目标：讲清楚——
1. 语言信息在网络内部如何编码
2. 已发现的数学结构是什么
3. 哪些结论跨模型稳健，哪些已被推翻
4. 当前瓶颈和下一步计划

**最核心的背景判断**：

1. 深度神经网络提取了某种高度压缩的语言数学结构
2. 这套结构不是"词典式存储"，而是"语义基底方向上的微小偏移"
3. 四模型（Qwen3/DS7B/GLM4/Gemma4）共享几何不变量，但编码策略模型特异
4. 研究路线：拼图积累 -> 稳定结构压缩 -> 因果打击 -> 跨模型复核

---

## 1. 研究原则

### 1.1 先拼图，后理论

1. 先积累可复核的实验拼图
2. 再压缩出稳定结构
3. 再做因果打击和跨模型复核
4. 最后才讨论第一性原理

**先机制，后定理；先还原，后抽象。**

### 1.2 先因果，后修辞

结论保留标准：
1. 能在原始激活或行为中直接看见
2. 能被消融或干预打中
3. 能跨样本复现
4. 能跨模型复现
5. 能被更强实验推翻或修正

### 1.3 先统一变量，后局部故事

1. 不把局部亮点误当成全局机制
2. 不把工程噪声误当成理论信号
3. 不把单个模型风格误当成普遍定律
4. 优先寻找统一控制变量

### 1.4 严格区分"描述"和"因果"

描述方程（精确描述发生了什么）≠ 因果方程（预测改变X后Y如何变）
当前已有精确描述方程，因果方程仍是薄弱环节。

## 2. 统一理论 v4.1（最新版本）

基于 P15-P43（Stage 662-689）全部实验结果。

### 2.1 五个公理

**第一公理：高维几何公理**（P28修正）
- 不同文本方向在高维空间趋向正交：E[|cos(Δ_l, h_l)|] ≈ O(1/√d)
- 需要d > d*（高维阈值，d* >> 512）
- 小模型(d=128): angle≈15°，不满足正交性
- 旋转角≈90°是高维空间的几何性质（INV-273, 4/4确认）

**第二公理：信息域公理**（P29/P35修正）
- 纠缠度 ≈ C · √(K_domain/d) · f(RL, data, arch)
- 单域模型（纯语言）→ SEPARATED策略
- 多域模型（语言+视觉+听觉）→ DENSE策略
- RL训练→能力融合（DS7B纠缠度最高，std最大）
- f(RL)使DS7B偏离理论预测16x

**第三公理：归一化隔离公理**
- 残存 ≈ (1-ρ_avg)^N_norms
- 归一化是隔离的核心机制（INV-305, 确认）
- Gemma4多模态模型的norm密度14.5/层(506个) >> 纯语言模型2-4/层

**第四公理：信号聚焦公理**（P28修正）
- 架构固有属性（训练开始即成立，聚焦比≈2.3）
- 后期层信号更强（INV-332, 4/4确认）
- 聚焦比 = cos_avg(后期)/cos_avg(前期) > 1.5

**第五公理：Logit精确公理**（P27确认）
- margin = ||h_L|| · ||Δw|| · cos(h_L, Δw)，误差<0.3%
- 数学恒等式（训练开始即成立）
- 之前"低估100倍"是方法论错误（avg_logprob vs 单步logit）

### 2.2 核心发现：语义基底方向（P43）

**这是当前最重要的结构性发现**：

- L0层所有文本方向完全对齐（cos=1.000）——共享初始方向
- 随深度发散：L0=1.000 → L_final=0.788-0.934
- 语义基底捕获82-94%方差（Qwen3最强93.8%，GLM4最弱81.6%）
- 残差（垂直于基底）仅编码6-18%
- **语言编码的本质**：在高度对齐的"语义基底方向"上，通过微小方向偏移和norm调制编码不同语义内容

| 模型 | 方向对齐 | PC1解释% | 基底捕获% | 残差编码% | Norm CV |
|------|---------|---------|-----------|---------|---------|
| Qwen3 | 0.934 | 24.1% | 93.8% | 6.2% | 3.3% |
| DS7B | 0.903 | 54.2% | 92.1% | 7.9% | 8.0% |
| GLM4 | 0.788 | 22.7% | 81.6% | 18.4% | 4.4% |
| Gemma4 | 0.880 | 19.9% | 89.3% | 10.7% | 22.5% |

### 2.3 方向+norm协同编码（P36）

- **方向承载几乎所有域区分信息**（Silhouette方向 > Norm）
- RMSNorm使norm域间差异仅2.5-2.7%（Qwen3/GLM4）
- DS7B例外：Norm域间差异19.5%，norm也携带大量信息

### 2.4 层动力学模型（P38）

| 模型 | 模式 | 域分离最强层 | 特征 |
|------|------|------------|------|
| Qwen3 | 倒拱形 | L8(0.14) | 早期峰值→缓慢衰减 |
| DS7B | U形 | L3(0.04) | L5-L26深度融合区(sil≈-0.21) |
| GLM4 | 平稳型 | L1(0.09) | 全程变化极小 |
| Gemma4 | U形(轻) | -- | 与DS7B类似但幅度小 |

### 2.5 信息瓶颈理论（P37）

- DS7B: PCA90=13维（超级压缩，Top1方差45.7%）
- Qwen3: PCA90=45维（均匀分布）
- 压缩度与纠缠度正相关

### 2.6 多步累积方程（P34）

- avg_logprob ≈ log(avg_margin) - C，C≈1.5（跨模型稳定常数）
- margin在多步中放大/稀释（模型依赖）
- "100倍低估"是线性vs对数尺度的自然差异

### 2.7 注意力-方向耦合（P41）

- DS7B: 强耦合(r=0.528)——注意力分布变化与方向变化强正相关
- Qwen3/GLM4/Gemma4: 弱耦合(r≈0.06)
- GLM4后期自关注增加(6%→19%)
- Gemma4注意力最不稳定(CV=30.7%)

### 2.8 层敏感性地图（P42）

| 模型 | 最敏感层 | delta | 模式 |
|------|---------|-------|------|
| Qwen3 | L15 | -13.5 | 拱形 |

### 3.2 Gemma4"例外"的统一解释（P26-P31多模态假说）

Gemma4不是"编码更差"，而是"编码不同"——它是多模态模型：

| 现象 | 多模态假说的解释 |
|------|----------------|
| 抵消率>90% | 多模态信号在共享空间中必然高抵消 |
| 信号效率13% | 大量维度用于视觉/听觉编码 |
| PCA90较高(16-22维) | 需编码视觉特征+语言特征 |
| Norm密度14.5/层(506个) | 模态内+模态间归一化 |
| 鲁棒性极高(3.86) | 多模态冗余带来的天然鲁棒性 |
| 方向对齐最高(0.972) | 多模态训练要求更强的基底对齐 |
| d_norm不增长(1.1x) | delta_angle=119°导致47.7%能量抵消 |

### 3.3 DS7B"深度融合区"（P38/P42）

- L5-L26共22层处于持续DENSE状态(sil≈-0.21)
- 恰好是推理能力最强的区域
- RL训练创造了"深度融合区"，让不同能力深度交互
- 中间层平均敏感性(-7.0) >> 边缘层(-2.3)（P42确认）
- 信息瓶颈：PCA90=13维，Top1=45.7%（P37）

## 5. 关键硬伤与瓶颈

### 5.1 当前最关键的未解决问题

1. **"语义基底方向"的存在性和唯一性未严格证明**——为什么所有文本在L0完全对齐？这个基底方向是由什么决定的？能否被精确计算？
2. **修正因子f(RL, data, arch)的数学形式未知**——纠缠度公式中最大的未知项
3. **多步生成方程缺失**——能精确描述单步决策(margin)，但不能预测多步生成过程
4. **RL训练如何改变"基底选择"策略**——DS7B的深度融合区是RL创造的，但机制不明
5. **跨架构编码拓扑不一致**——Qwen3-Gemma4编码距离r=0.03，不同架构几乎是不同的"语言"
6. **因果效应量偏小**——跨任务因果子集效用0.001-0.025，方向正确但效应弱
7. **推理能力分布式编码**——恢复率<4%，无法用低维捕捉
8. **文本→方向偏移的可预测映射缺失**——知道"基底方向"存在，但不知道如何从文本内容预测偏移量

### 5.2 方法论限制

1. **Gemma4 hook机制不兼容**——架构差异导致部分消融实验无法执行
2. **GPU内存限制**——四模型必须串行测试
3. **单token vs 多token的方法论鸿沟**——avg_logprob vs 单步logit差7.4x
4. **高维阈值d*未精确定位**——已知d*>512，但精确值未知

### 6.2 更宏观的突破方向

**方向A：从"描述"到"预测"**
- 当前能精确描述hidden state的结构，但不能从输入预测输出
- 需要建立"文本→基底偏移→margin→概率"的完整预测链
- 关键瓶颈：文本→方向偏移的映射函数

**方向B：RL训练的编码效应**
- DS7B是唯一经过RL训练的模型，展现独特的"深度融合区"
- 需要对比：同架构模型在有/无RL训练时的编码差异
- 核心问题：RL是否创造了新的数学结构，还是只是强化了已有结构？

**方向C：跨架构不变量**
- 当前不变量在跨架构比较中大量失效
- 需要回答：哪些是"Transformer的不变量"，哪些是"特定架构的不变量"？
- 可能需要在更多架构（Mamba、RWKV等）上验证

**方向D：信息域的数学定义**
- "信息域"目前只是操作化概念，缺乏严格数学定义
- 可能的数学工具：信息几何、拓扑数据分析、流形学习
- 目标：从hidden state的几何结构中自动发现信息域

## 9. 当前阶段最严格的表述

如果只保留一段最严格的话：

> **所有文本在L0层共享完全相同的"语义基底方向"（cos=1.0），随深度发散到0.79-0.93。语言编码的本质是在这个基底方向上通过微小偏移和norm调制区分语义。基底捕获82-94%方差，残差仅6-18%。方向承载几乎所有域区分信息（RMSNorm使norm域间差异仅2.5%）。DS7B通过RL训练创造了"深度融合区"（L5-L26, PCA90=13维），将信息极端压缩到超级维度。Logit方程完全精确（margin=cos(h,u)×||h||×||u||, 误差<0.3%），多步累积遵循avg_logprob≈log(margin)-1.5。四模型共享几何不变量（旋转≈90°, 抵消有序, 信号聚焦），但编码策略模型特异（纠缠度0.065-0.282, 基底捕获82-94%）。Gemma4的多模态架构解释了其"例外"——更高的norm密度和更低的信号效率是多模态编码的必要代价。**
