## 三、当前最核心的5个未解问题

### 问题1：模板信号到底是什么？（占残差流90%+）
- **已知**: 减去模板均值后类别信号暴露，但模板本身的数学结构完全未知
- **未知**: 模板是否包含语言共性信息？还是仅仅是句式？
- **PUZZLE对应**: GR-1a (语法编码)
- **突破路径**: 需要新的分析工具——不能只减均值，需要对模板做谱分解

### 问题2：概念编码是否具有跨任务因果性？
- **已知**: PCA5探针0.98-1.00，但深层干预无效
- **未知**: 中间层干预是否有效？概念编码在多大程度上是因果载体？
- **PUZZLE对应**: KN-1d (稀疏vs稠密), UN-1 (概念×逻辑)
- **突破路径**: 中间层干预实验 + 跨任务概念追踪

### 问题3：逻辑推理的编码载体是什么？
- **已知**: 逻辑信号强(2.7x属性)，编码在正交方向，L18峰值
- **未知**: 逻辑推理的具体计算机制——注意力头？FFN？组合？
- **PUZZLE对应**: LG-1a (逻辑编码位置), LG-2a (逻辑计算机制)
- **突破路径**: 注意力头级别因果追踪 + 逻辑推理消融实验

### 问题4：知识、逻辑、语法是否共享同一数学基底？
- **已知**: 三者编码在不同正交子空间，但可能有共享的结构
- **未知**: 是否存在跨能力的统一编码基底？还是完全独立？
- **PUZZLE对应**: UN-1 (概念×逻辑共享编码), UN-2 (语法×语义)
- **突破路径**: 跨任务SAE特征复用率分析

### 问题5：如何从"统计描述"升级到"可预测理论"？
- **已知**: 大量统计量(cos, norm, probe accuracy)但无法从文本预测编码
- **未知**: 给定一个新句子，能否预测其残差流编码？
- **PUZZLE对应**: 全局——从5-7%解释力提升到>80%
- **突破路径**: 先解决问题1-4，积累足够拼图后尝试构建预测方程

## 五、近期实验路线图

### Phase CCLXI：中间层因果干预（优先级P0）
- **目标**: 验证中间层(L10-L20)的概念替换是否有效
- **如果有效**: 概念编码是因果载体，可构建因果理论
- **如果无效**: 概念编码是衍生现象，需从注意力模式寻找数学结构

### Phase CCLXII：逻辑推理消融（优先级P0）
- **目标**: 消融L18附近注意力头，观察逻辑推理能力变化
- **预期**: 找到逻辑推理的因果载体（注意力头组合）

### Phase CCLXIII：模板信号谱分解（优先级P1）
- **目标**: 对残差流90%的模板信号做PCA/ICA分解，分析其内部结构
- **预期**: 模板信号可能包含语言共性信息，不仅仅是句式

### Phase CCLXIV：跨任务SAE特征复用（优先级P1）
- **目标**: 用SAE分解残差流，测量同一特征在知识/逻辑/语法任务中的复用率
- **预期**: 如果复用率>30%，说明存在跨能力的统一编码基底
