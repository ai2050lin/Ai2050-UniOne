## 0. 总览：语言在 DNN 中的完整运行图

```
输入文字 "红色的苹果"
   ↓ Tokenizer (分词器)
Token IDs: [3567, 18923, 4561]
   ↓ W_E 嵌入 (Embedding)
h_0 = W_E[token] + W_pos[位置]     ← 59%能量前10分量，各词独立编码
   ↓ L0→L1 非线性断裂 (92%旋转在此一步)
h_1                                ← cos(L0,L1)=0.14, 变成分布式编码
   ↓ 中间层信号重编码 (×36层)
h_l = G_l · R_l · h_{l-1} + ε_l   ← LayerNorm放大, 结构化旋转
   · 信号逐层偏离注入方向 (cos→0)
   · 信号能量逐层向W_U行空间转移
   · recoding_ratio = ratio(k) 随SVD截断k变化!
   ↓ 最后一层
h_final                            ← 投影到W_U空间的比例 = ratio(k_max)≈1.0
   ↓ W_U 解码 (Unembedding)
logits = h_final @ W_U^T           ← 151936词几乎正交 (cos≈0.004)
   ↓ softmax
概率分布 → 选最高概率词 → "苹果"
```

**与之前理解的核心差异**：

| 项目 | 之前理解 (已推翻) | 修正后理解 |
|------|-----------------|-----------|
| recoding_ratio | ≈0.22 恒定不变量 | ratio(k): 随 SVD 截断 k 从 0.15→1.0 增长 |
| "2×因子" | recoding_ratio ≈ 2×PR/d, 未解之谜 | 截断伪影，k=300 低估了 PR |
| 信号传播模型 | 各向同性×随机旋转×固定比例 | 各向同性×结构化旋转×频谱相关比例 |
| W_U 空间利用 | 只利用了22% | 几乎完全利用 (k=max 时 ratio→1.0) |
| ratio 的本质 | 统计不变量 | 信号在 W_U 奇异向量上的频谱分布函数 |

## 1. 第零步：输入——文字变成向量 (Embedding)

### 1.1 发生了什么

当你输入"红色的苹果"时：

1. **Tokenizer** (分词器) 把文字切成 token：`["红", "色的", "苹果"]`
2. **查 W_E** (嵌入矩阵)：每个 token ID → d_model 维向量
   - W_E shape = [vocab_size, d_model]，15万词各有一个2560维"身份证"
3. **加位置编码** W_pos：让模型知道词序
4. **结果**：`h_0 = W_E[token] + W_pos[位置]`

### 1.2 关键实验发现

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| Embedding 59%能量前10分量 | P440 | 极度各向异性 |
| 各词嵌入几乎正交 | P435 | cos(词i, 词j)≈0.04 |
| 位置编码占<3%能量 | P440 | W_pos 范数远小于 W_E |

### 1.3 推理过程

1. 对 W_E 的各行做 SVD，统计累积能量分布
2. 发现前10个奇异值占了59%的能量 → 存在"强方向"和"弱方向"
3. 计算词嵌入间的cos相似度 → ≈0.04 → 各词独立编码
4. 这符合 Johnson-Lindenstrauss 引理：高维空间天然使随机向量几乎正交

### 1.4 含义

Embedding 空间不是均匀的。存在"强方向"（前10个奇异向量）承载主要语义信息，"弱方向"承载精细区分。这为后续的信号重编码提供了基础——强方向是信号注入的入口，弱方向是逃逸的通道。

## 2. 第一步：L0→L1——非线性断裂层 (整个网络最关键的一步)

### 2.1 发生了什么

从嵌入空间到表示空间的根本性转换。L0→L1 不是简单的线性变换，而是"维度的重新组织"。

### 2.2 关键实验发现

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| L0→L1 的 cos 仅0.14 | P47 | 远低于其他层(L1→L2≈0.9) |
| 92%的方向旋转集中在L0→L1 | P47/P49 | |
| MLP无法拟合L0→L1映射 | P246 | R²仅0.02(GLM4) |
| L0的attn占delta的87-94% | P237 | attn主导信号注入 |
| 残差连接精确成立 | P237 | h_L = h_{L-1} + attn + ffn, 误差≈0 |

### 2.3 推理过程

1. **P47**：逐层计算相邻层的 cos 相似度，发现 L0→L1 出现断崖式下降 (cos≈0.14)
2. **P49**：计算逐层累积旋转角度，发现 92% 集中在 L0→L1
3. **P246**：尝试用 MLP 拟合 h_0→h_1 映射，R²=0.02 → 非线性极强，简单函数无法描述
4. **P237**：验证残差连接，确认 h_1 = h_0 + attn_0(h_0) + ffn_0(h_0)

### 2.4 含义

L0→L1 是"从嵌入空间到表示空间"的根本转换：
- Embedding 空间中每个词独立正交 (cos≈0.04)
- 经 L0→L1 后变成分布式编码 (cos≈0.63)
- 这不是逐层渐进的，而是**一步跳变**
- 类比：就像从"二维照片"转换到"三维模型"——维度相同(都是2560维)，但空间结构完全不同

**未解问题**：L0→L1 的非线性形式是什么？MLP 无法拟合 (R²=0.02)，可能需要新数学工具。

## 3. 第二步：中间层——信号重编码 (核心过程)

### 3.1 发生了什么

信号从注入方向（如"红色"方向）逐层传播，方向不断偏离，能量不断被重新分配到 W_U 行空间中。这是语言信息从"语义空间"向"词汇空间"转换的核心过程。

### 3.2 关键实验发现 (含 P452 修正)

#### 3.2.1 信号逐层偏离注入方向

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| 信号方向与注入方向的cos逐层衰减 | P438 | cos(inject, Δ_l) 从 L0 的 ≈1 → L35 的 ≈0 |
| 99.6% 的信号逃逸了 V_lang | P438 | 注入的"红色"方向，到最后一层只剩 0.3% |

#### 3.2.2 信号能量向 W_U 行空间转移

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| hidden state 在 W_U 空间中投影比从 0% 增长到 72% | P436 | |
| 信号 Δ 在 V_lang 和 W_U 中此消彼长 | P438 | |

#### 3.2.3 ★ P452 核心修正：recoding_ratio 是 k 的函数 ★

**之前的错误理解**：

#### 3.2.5 三空间独立性

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| 权重/激活/信息流空间几乎独立 | P442 | 所有 cos≈0.02 |

### 3.3 推理过程

**(1) 发现 99.6% 逃逸** (P435→P438)：
- P435：对 W_U 做 SVD，发现 PR=249-395 (k=300时)，覆盖 >8% 的 d_model
- P436：逐层测量 hidden state 在 W_U 空间中的投影比，发现从 0% 增长到 72%
- P438：同时测量 Δ 在 V_lang 和 W_U 中的占比，发现此消彼长

**(2) 发现 ratio 依赖 k** (P451→P452)：
- P451 (Phase XCII)：对 W_U 做多尺度 SVD，发现 PR 随 k 快速增长
  - k=300: PR=219, PR/d=0.086
  - k=2558: PR=2285, PR/d=0.892 → W_U 几乎满秩!
- P452：用不同 k 重测 ratio，发现 ratio 随 k 单调增长
  - Qwen3: ratio(300)=0.15, ratio(2558)=0.998
  - GLM4: ratio(300)=0.19, ratio(1500)=0.41

**(3) 推翻"0.22恒定"假说**：
- 之前所有 ratio 测量都用 k=300 截断 SVD
- k=300 时 ratio≈0.15-0.19，不是0.22
- Phase XC 用 k=500 时 ratio≈0.22-0.24
- **"0.22"只是 k=500 时的特定值，不是不变量**

**(4) 发现三空间独立** (P442)：
- 测量权重空间(W)、激活空间(A)、信息流空间(I)的交叉 cos
- 全部≈0.02 → 信号传播是高度非线性的旋转+缩放

### 3.4 含义

**之前的错误理解**：
```
信号注入V_lang方向 (L0: 100%在V_lang)
  → 随机旋转 (cos→0)
  → 固定比例22%进入W_U空间
  → recoding_ratio≈0.22 是统计不变量
```

**修正后的理解**：
```
信号注入V_lang方向 (L0: 100%在V_lang)
  → 结构化旋转 (cos→0, 但不是纯随机)
  → 信号频谱分布在W_U奇异向量上
  → ratio(k) = 信号在前k个奇异向量上的投影能量占比
  → ratio(k_max)→1.0 (W_U几乎满秩，信号几乎完全在W_U空间中)
```

**核心差异**：
1. 信号没有"浪费"——72% 的 hidden state 能量在 W_U 空间中 (k=300 时只能捕获15%)
2. ratio 不是不变量——它是频谱分布函数，取决于你观察的"分辨率" k
3. "99.6%逃逸V_lang"是真实的，但"逃逸"不等于"丢失"——能量转移到了 W_U 空间

### 3.5 未解问题

1. **ratio(k) 的函数形式为什么不同？** Qwen3 是幂律，GLM4 是指数饱和
2. **信号在 W_U 奇异向量上的能量分布是什么？** 集中在大奇异值方向？均匀分布？
3. **ratio(k) 的跨层变化** — 不同层的 Δ 在 W_U 空间中的频谱分布是否不同？
4. **DeepSeek7B 的 P452 数据缺失** — 需要补测
5. **因果链前半段(ΔW→Δh)不可预测** — gain 与所有当前权重特征无关
6. **gamma_energy 与 gamma_info 负相关** — 能量聚焦≠信息聚焦，桥梁缺失
7. **DS7B 传播机制根本不同** — R²=0.30，93%层不稳定，需要覆盖不稳定模型的统一理论
8. **post_ln_norm 的物理角色** — 控制残差增量绝对量，因果链成立(Qwen3)，DS7B 断裂
9. **W_gate 是谱尖锐度来源** — PR最低(0.58-0.66)，MP偏离最大(DS7B=962)
10. **L0 首层极不稳定** — ratio=54-5000，嵌入层输出太小(h_in=0.3-4.5)

## 4. 第三步：信号放大——LayerNorm 是绝对主因

### 4.1 发生了什么

信号在传播过程中被逐层放大。放大机制不是来自 MLP 或 Attention，而是来自 LayerNorm/RMSNorm。

### 4.2 关键实验发现

| 发现 | 实验编号 | 数据 |
|------|---------|------|
| LayerNorm增益远大于Attn/MLP | P425 | GLM4: LN=827.8, Attn=0.016, MLP=0.037 |
| MLP和Attn压缩信号 | P425 | 它们的输出范数远小于残差 |
| 信号放大 = LayerNorm × 多层累积 | P425 | 每层1-3倍，36层=指数增长 |

各模型 LayerNorm 增益：

| 模型 | LayerNorm增益 | Attn增益 | MLP增益 |
|------|-------------|---------|--------|
| Qwen3 | **59.2** | 0.39 | 0.76 |
| GLM4 | **827.8** | 0.016 | 0.037 |
| DS7B | **43.9** | 1.83 | 2.20 |

### 4.3 推理过程

1. 直觉：MLP 和 Attn 应该放大信号？**实测相反！**
2. P425 分别测量 LN/attn/mlp 对信号范数的影响
3. 发现 LN 的增益公式：`gain = 1/std(residual)`
   - 残差范数小时，gain 极大 (GLM4: 827.8)
   - 残差范数大时，gain 很小
4. MLP/Attn 的输出范数远小于残差 → 它们是"压缩"而非"放大"

### 4.4 含义

LayerNorm 的作用类似**自动增益控制** (AGC)：
- RMSNorm 把大范数压缩到固定范围，但等比例放大了信号Δ的相对比例
- 类比：把房间(残差)缩小了，但信号(人)没变，所以信号在房间中的"占比"变大了
- 这是信号被"看到"的关键机制——不是信号本身变大了，而是背景变小了

## 5. 第四步：信号传播方程——统计力学框架

### 5.1 推导过程

**已知事实**：

1. P419：线性 Jacobian 只解释 1-27% 的变化 → **线性理论失败**
2. P442：三空间独立 (cos≈0.02) → **不存在确定性映射**
3. P452：ratio(k) 不是常数 → **不是简单统计不变量**

三个事实共同指向：信号传播不能简单用线性模型或纯随机模型描述。

**修正后的信号传播方程**：

```
h_l = G_l · R_l · h_{l-1} + ε_l

G_l = 逐层增益 (LayerNorm主导)
R_l = 结构化旋转矩阵 (不是纯随机!)
ε_l = 噪声/非线性修正项

recoding_ratio(k) = ||U_k^T · Δ_l||² / ||Δ_l||²

其中 U_k 是 W_U^T 的前k个左奇异向量
```

### 5.2 "结构化旋转"vs"随机旋转"

**之前假设**：R_l 是 Haar 随机旋转 → ratio = k/d → 纯几何约束

**P452 修正**：
- 如果是纯随机旋转 → ratio(k) = k/d (线性)
- Qwen3 实测：ratio(k) ≈ 0.992·(k/2560)^0.903 — 接近线性但略有弯曲
- GLM4 实测：ratio(k) ≈ 0.884·(1-e^{-0.000267k}) — 明显非线性

**推理**：
- Qwen3 的 ratio(k) 接近 k/d → 旋转接近纯随机 → 信号频谱接近均匀分布
- GLM4 的 ratio(k) 是指数饱和 → 旋转有结构 → 信号集中在前几个大奇异值方向
- 不同模型的"旋转结构"不同 → 这可能是训练策略/架构差异的结果

### 5.3 统计力学类比 (修正)

**之前的类比**：recoding_ratio ≈ 0.22 类似温度——统计不变量

**修正后的类比**：ratio(k) 类似**频谱密度函数**——不是不变量，而是分布函数

```
ratio(k) = ∫₀ᵏ S_Δ(ω) dω

其中 S_Δ(ω) 是信号Δ在W_U奇异值方向上的能量密度
```

- 如果 S_Δ(ω) = 常数 → ratio(k) = k/d → 纯随机
- 如果 S_Δ(ω) ∝ ω^(-α) → ratio(k) = 幂律
- 如果 S_Δ(ω) 集中在低频 → ratio(k) = 指数饱和

## 6. 第五步：P451/P452 修正——从"2×因子"到"频谱分布"

### 6.1 "2×因子"的来龙去脉

**之前的困惑**：
```
PR_W_U(k=300) / d_model ≈ 219/2560 ≈ 0.086
实测 ratio(k=300) ≈ 0.15
0.15 / 0.086 ≈ 1.74倍 → "2×因子"
```

**P451 的解释**：
- PR 本身就是 k 的函数
- k=300 时 PR=219, k=2558 时 PR=2285
- PR/d 从 0.086 增长到 0.892

**P452 的进一步确认**：
- ratio(k=300) ≈ PR(k=300)/d × 修正因子
- 但修正因子不是固定2×，而是随 k 变化
- k=500 时：ratio=0.235, PR/d=0.154, 修正因子=1.53
- k=800 时：ratio=0.348, PR/d=0.261, 修正因子=1.33
- k=2558 时：ratio=0.998, PR/d=0.892, 修正因子=1.12

**结论**："2×因子"是 k=300 时的偶然现象，不是物理效应。

### 6.2 ratio(k) ≠ PR(k)/d 的含义

即使 k=max，ratio→1.0 而 PR/d→0.892，仍然有差距。这意味着：
- ratio(k) 严格大于 PR(k)/d（信号不只是均匀分布在奇异值方向上）
- 信号能量在大奇异值方向上略有过剩 → ratio(k) > k/d
- 这个"过剩"随 k 减小 → 在 k=max 时 ratio→1.0

## 13. 附录：对"未达饱和"假说的严格检验 (2026-04-13)

> 有一种直觉分析认为：P451 发现 ratio(k=800)=0.22 < PR/d(k=800)=0.26，说明信号"未达饱和"，
> 即 W_U 提供了 0.26 的容量但信号只占了 0.22，填充率 84.6%。
> 这种分析**看似合理但数学上有严重错误**。

### 13.1 错误根源：跨 k 值比较

那段分析混淆了两个不同 k 值下的测量：

- **ratio** 用 k=300 测量（之前 P443-446 的标准设置）
- **PR/d** 用 k=800 计算（P451 新数据）

但 **ratio 和 PR 必须用同一个 k**！因为 ratio(k) = 信号在前k个奇异向量上的投影比，PR(k) = 前k个奇异值的有效维度。

### 13.2 正确的同 k 值比较 (P452 实测)

Qwen3 (d_model=2560):

| k | ratio(k) | PR(k)/d | ratio/PR·d | ratio - PR/d |
|---|----------|---------|-----------|-------------|
| 300 | 0.150 | 0.086 | **1.74** | +0.064 |
| 500 | 0.235 | 0.154 | **1.53** | +0.081 |
| 800 | 0.348 | 0.261 | **1.33** | +0.087 |
| 1200 | 0.496 | 0.406 | **1.22** | +0.090 |
| 1500 | 0.600 | 0.515 | **1.16** | +0.085 |
| 2558 | 0.998 | 0.892 | **1.12** | +0.106 |

**关键发现**：在所有 k 值下，ratio(k) > PR(k)/d！差额约 0.06-0.11。

### 13.3 "ratio > PR/d"的正确物理解释

PR/d 是**奇异值分布的集中度**——如果所有奇异值相等，PR=k，PR/d=k/d。

ratio > PR/d 意味着：**信号在前k个奇异向量上的投影能量占比，大于这k个奇异值按能量计算的"公平份额"**。

等价说法：**信号能量略偏大奇异值方向**（大奇异值方向上的信号密度 > 小奇异值方向上的信号密度）。

这与"未达饱和"恰恰相反——信号不是"没填满"大奇异值方向，而是"偏重"了大奇异值方向。

### 13.4 "0.22"不是常数

| k | ratio(k) |
|---|----------|
| 300 | 0.150 |
| 500 | 0.235 ≈ "之前的0.22" |
| 800 | 0.348 |
| 2558 | 0.998 |

"0.22"只是 k=500 时的特定测量值，不是物理不变量。不存在"0.22是高维稳态解"——k变了，ratio就变了。

### 13.5 "随机旋转后投影=0.22"需要重新推导

如果信号经纯随机旋转后投影到 k 维子空间：
```
ratio_random(k) = k / d_model  (各向同性假设)
```

Qwen3 实测：
```
ratio(k) ≈ 0.992 × (k/2560)^0.903
```

- 指数 0.903 < 1 → ratio(k) > k/d → 信号**略偏**大奇异值方向
- 偏离很小（0.992 vs 1.0, 0.903 vs 1.0）→ 接近随机，但有弱结构
- 不是纯随机旋转，但也不是强聚焦

### 13.6 正确的物理图景

**错误图景（"未达饱和"）**：
```
W_U 提供了 26% 的容量 (PR/d=0.26, k=800)
信号只占了 22% (ratio=0.22, 但这是k=300的!)
→ 信号"未填满" → 78%的空间闲置
```

**正确图景（"频谱分布"）**：
```
W_U 行空间几乎覆盖整个 d_model (k=max 时 ratio→1.0)
信号在不同 k 分辨率下有不同的投影比
  - k=300: 只能看到15% (低分辨率)
  - k=800: 能看到35% (中分辨率)
  - k=2558: 能看到100% (高分辨率)
→ 信号没有被"浪费"，只是低分辨率时看不到
→ ratio(k) 的函数形式 = 信号频谱密度的累积分布
```

### 13.7 为什么 ratio(k) > k/d 有意义？

差额 ratio(k) - k/d ≈ 0.06-0.11，说明：
1. 信号经多层传播后，不是完全各向同性的
2. LayerNorm 的增益调控可能略微偏向大奇异值方向
3. 这个"弱偏置"在所有 k 值下稳定存在（差额≈0.08左右）
4. 这可能是 LayerNorm 的"归一化→等权重→略偏大奇异值方向"的结果

### 13.8 对"几何灵魂"假说的评估

原问题："在 d=2560、步角80°的条件下，经过36次旋转后，向量在一个 PR=667 的子空间上的能量投影期望是否等于 0.22？"

**修正后的问题**："在 d=2560 的空间中，信号经36层结构化旋转后，其频谱密度 S_Δ(ω) 是什么？由此推导的 ratio(k) = ∫₀ᵏ S_Δ(ω) dω 是否与实测幂律 0.992·(k/2560)^0.903 一致？"

**评估**：
- 纯随机旋转 → ratio(k) = k/d → 幂律指数=1.0 → 与实测0.903不符
- 实测指数 0.903 < 1 → 旋转不是纯随机的
- 需要定量推导：什么样的"弱结构旋转"能产生指数0.903的幂律？
- 这确实是"几何灵魂"级别的深刻问题，但答案不是简单的"随机旋转=0.22"
