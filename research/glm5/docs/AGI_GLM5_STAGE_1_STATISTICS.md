# AGI 研究阶段分析报告：阶段 1

## 1. 阶段概述：现象学统计与静态观测
本阶段旨在通过对大规模预训练模型（如 GPT-2、Qwen 系列）的静态参数与激活状态进行统计，挖掘神经网络隐藏的几何与物理特性。

## 2. 测试列表
*   **L0 稀疏度统计**：跨层测量神经元激活的非零占比。
*   **权重矩阵正交性测试**：利用 SVD（奇异值分解）测量 W_embed 与 W_U 的行/列正交性得分。
*   **层级抽象能力评估**：通过对 Layer 0 到 Layer 11 的表征进行对比，测量语义抽象度的递增比例。
*   **编码关键层识别**：识别在整个编码任务中贡献度最高的神经层（初步锁定 Layer 2, 6, 10, 11）。

## 3. 核心发现列表
*   **稳定稀疏性**：在大规模模型中，平均 L0 稀疏度稳定在 **78%** 左右，显示了极其高效的信息压缩。
*   **高正交性**：输出解码空间 (W_U) 的正交性得分高达 **97%**，这使得不同概念在几何上能够解耦。
*   **层级加速**：Layer 0 到 Layer 11 表现出约 **10%** 的抽象能力阶梯式提升。
*   **5 大神经科学类比**：
    1. 稀疏编码阈值机制
    2. 高维空间天然正交性
    3. 侧向抑制竞争学习
    4. Hebb 学习驱动的特征涌现
    5. 稀疏激活带来的能效优化

## 4. 重要结论
*   **几何决定论**：DNN 的表征空间不是混沌的，而是由高度有序的稀疏正交基组支撑的。
*   **计算效率**：稀疏性不仅是生物演化的结果，也是人工神经网络在梯度优化下达到互信息最大化的物理选择。
*   **抽象分层**：智能的演进逻辑初步显现在层级间的偏转与放大模式中。

## 5. 主要问题 (Hard Flaws)
> [!CAUTION]
> ### 第一性原理缺失
> 虽然观测到了 78% 稀疏度和 97% 正交性，但完全无法解释**为什么**是这些具体数值。
> 
> ### 缺乏预测能力
> 当前发现仅属于“事后解释”，无法指导如何在训练前通过调整超参数来控制这些指标。
> 
> ### 动力学断层
> 只看到了训练后的静态结果，完全不知道这些特性是在训练的哪一个具体步数、通过何种机制“涌现”出来的。
