# AGI 研究阶段分析报告：阶段 3

## 1. 阶段概述：结构分解与语言编码架构
本阶段致力于将复杂的语言能力解构为可理解的物理维度。研究重点从整体统计转向局部功能定位，确立了语言在残差流中的分层叠加架构。

## 2. 测试列表
*   **三层编码定位实验**：测量语法 (L0-8)、逻辑 (L4-20) 和风格 (L12-32) 在残差流中的分布范数。
*   **基底-偏置 (Basis-Bias) 验证**：对 132 个概念进行残差流投影，计算类别基底解释方差 (R²)。
*   **SVD 语义因子分解**：对 220 个概念的偏置矩阵进行 SVD 分解，提取可解释因子（如颜色、大小、领域）。
*   **ANOVA 属性关联测试**：计算语义因子与已知属性（如 fruit.color）之间的 Eta² 关联强度。
*   **跨模型概念算术**：测试 (dog - cat) 等算术在 Qwen 与 DeepSeek 之间的命中率。

## 3. 核心发现列表
*   **三层叠加架构**：语言不是独立模块，而是同一残差流上三种尺度的扰动：
    *   **语法层**：最局部、最硬，由 Induction Heads 驱动。
    *   **逻辑层**：最复杂、最分布，依赖 dim 138 (共享承载) 与 dim 5 (偏置偏转)。
    *   **风格层**：最软、最全局，表现为极低维的“调色旋钮”。
*   **线性为主**：实验证明 Autoencoder (非线性) 提取效果差于 SVD (线性)，确认概念偏置空间本质上是低维线性结构。
*   **语言无关性**：SVD 能够起到“语言去噪”的作用，中英文概念在因子层面的相关性高达 **0.75**，揭示了超越具体语言的数学结构。

## 4. 重要结论
*   **计算不变量**：虽然模型架构不同，但对于人类知识的编码都收敛到了高度一致的线性子空间中。
*   **功能对齐**： dim 138 是类别基底的几何重心，而 dim 5 则控制了逻辑方向的切换。
*   **重编码规律**：信号在传播过程中会经历从 V_lang 空间向 W_U 解码空间的“逃逸”，伴随着巨大的幅度放大（可达 800 倍）。

## 5. 主要问题 (Hard Flaws)
> [!IMPORTANT]
> ### 逻辑恢复率极低 (Logic Gap)
> 逻辑推理维度对输出预测的影响极小（JS 散度仅 ~0.02），逻辑恢复率不足 **4%**，主流推理机制依然由“暗物质”驱动。
> 
> ### 方差解释度天花板
> 即使使用前 20 个 SVD 因子，也只能解释约 **22%** 的空间方差，剩余 78% 的信息表现为噪声或不可理解的高维结构。
> 
> ### 分类均匀性缺失
> 语法混乱对注意力机制的影响极大（L1>0.01），但逻辑有效性几乎不改变注意力模式（L1<0.0002），说明现有分析工具对深层逻辑计算极不敏感。
