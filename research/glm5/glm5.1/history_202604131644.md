## W_U（Unembedding Matrix，反嵌入矩阵）

### 在 Transformer 中的位置

Transformer 语言模型的核心数据流：

```
输入文本 → Tokenizer → Token IDs → W_E (Embedding) → Transformer Layers → W_U (Unembedding) → Logits → 概率分布 → 下一个词
```

W_U 是模型的**最后一层**，负责把隐藏状态（hidden state）映射回词表空间。

### 具体定义

- **W_E**（Embedding Matrix）：shape = `[vocab_size, d_model]`，把 token ID 映射到 d_model 维向量
- **W_U**（Unembedding Matrix）：shape = `[vocab_size, d_model]`，把 d_model 维隐藏状态映射到 vocab_size 维 logits

```
logits = h_final @ W_U^T     # h_final: [1, d_model], W_U^T: [d_model, vocab_size]
                               # logits: [1, vocab_size]
```

W_U 的第 i 行就是词表中第 i 个词在 d_model 维空间中的"代表向量"。

### 在本项目中的关键意义

1. **W_U 行空间 = 语言可表达的空间**：W_U 有 151936 行（Qwen3 词表大小），但只有 2560 列（d_model），所以 151936 个词的向量全部挤在 2560 维空间中

2. **recoding_ratio 的定义**：信号 Δ（delta）在 W_U 行空间中的投影能量占比
   ```
   recoding_ratio = ||U^T · Δ||² / ||Δ||²
   ```
   其中 U 是 W_U 行空间的正交基（通过 SVD 得到）

3. **为什么研究 W_U**：如果一个信号在 W_U 行空间中无投影，那这个信号对输出词的预测**完全没有影响**。recoding_ratio≈0.22 意味着约 22% 的信号能量能影响输出，78% 被浪费了

4. **之前的关键发现**：
   - W_U 的 151936 行在 2560 维空间中**几乎正交**（cos≈0.004），这是 Johnson-Lindenstrauss 引理的体现
   - 词嵌入有数学结构（国王+王后-男性=女性），这些结构就编码在 W_U 的行向量中
   - W_U 接近满秩（PR/d≈0.89），说明 2560 维空间被词表向量几乎完全覆盖

### 代码中的获取方式

    return model.W_U.numpy()  # TransformerLens 提供的接口
```

## 语言在深度神经网络中的运行原理——LCS框架系统讲解

### 第零步：输入——文字变成向量（Embedding）

输入"红色的苹果"时：

1. **Tokenizer**（分词器）切成 token：`["红", "色的", "苹果"]`
2. **查 W_E**（嵌入矩阵）：每个 token ID → d_model 维向量
   - W_E shape = [151936, 2560]，15万词各有一个2560维"身份证"
3. **加位置编码** W_pos：让模型知道词序
4. **结果**：`h_0 = W_E[token] + W_pos[位置]`

**关键发现**（P440）：Embedding 层 59% 能量集中在前10个分量，极度各向异性——存在"强方向"和"弱方向"。

**推理方法**：对 W_E 的各行做 SVD，统计累积能量分布，发现前10个奇异值占了59%的能量。

---

### 第一步：L0→L1——非线性断裂层（整个网络最关键的一步）

**现象**：
- cos(h_0, h_1) = 0.14（几乎垂直！）
- 而 cos(h_1, h_2) ≈ 0.9（后续层变化很小）

**怎么分析出来的**：
1. P47 逐层计算相邻层的 cos 相似度，发现 L0→L1 出现断崖式下降
2. P246 尝试用 MLP 拟合 L0→L1 映射，R² 仅 0.02（GLM4），说明非线性极强
3. P237 验证残差连接精确成立：h_L = h_{L-1} + attn + ffn
4. P237 发现 L0 的 attn 占 delta 的 87-94%

**推理**：L0→L1 是"从嵌入空间到表示空间"的根本转换。Embedding 空间中每个词独立正交（cos≈0.04），但经 L0→L1 后变成分布式编码（cos≈0.63）。这不是逐层渐进的，而是**一步跳变**——92%的方向旋转集中在这一步。

---

### 第二步：中间层——信号重编码（核心过程）

**现象**：
- 注入"红色"方向的信号，到最后一层只剩0.3%在原始方向
- 但72%的能量进入了 W_U 行空间——信号没有丢失，只是换了"频道"
- recoding_ratio ≈ 0.22 恒定（三模型一致）

**怎么分析出来的**：

**(1) 发现 99.6% 逃逸**（P435→P438）：
- P435：对 W_U 做 SVD，发现 PR=249-395，覆盖 >89% 的 d_model
- P436：逐层测量 hidden state 在 W_U 空间中的投影比，发现从 0% 增长到 72%
- P438：同时测量 delta 在 V_lang 和 W_U 中的占比，发现此消彼长

**(2) 发现 recoding_ratio 恒定**（P439）：
- 定义：`recoding_ratio = ||P_{W_U}(Δ)||² / ||Δ||²`
- 对8个属性×12层×3模型测量，ratio 都≈0.22

**(3) 发现三空间独立**（P442）：
- 测量权重空间(W)、激活空间(A)、信息流空间(I)的交叉 cos
- 全部≈0.02，接近随机水平
- 推理：信号传播是高度非线性的旋转+缩放

**推理链**：
```
信号注入V_lang方向(L0: 100%在V_lang)
  → 逐层随机旋转(cos→0, 方向偏离注入方向)
  → 逐层信号放大(LayerNorm主导)
  → 信号"散射"到所有方向
  → 72%恰好落在W_U行空间(可被解码)
  → 28%在W_U空间外(对输出无贡献)
  → recoding_ratio ≈ 72% × (W_U覆盖/总维度) ≈ 0.22
```

---

### 第三步：信号放大——LayerNorm 是绝对主因

**反直觉发现**（P425）：MLP 和 Attention **压缩**信号，不是放大！

| 模型 | LayerNorm增益 | Attn增益 | MLP增益 |
|------|-------------|---------|--------|
| Qwen3 | **59.2** | 0.39 | 0.76 |
| GLM4 | **827.8** | 0.016 | 0.037 |
| DS7B | **43.9** | 1.83 | 2.20 |

**推理过程**：
1. 直觉：MLP 和 Attn 应该放大信号？实测相反！
2. P425 分别测量 LN/attn/mlp 对信号范数的影响
3. 发现 LN 的增益公式：`gain = 1/std(residual)`
   - 残差范数小时，gain 极大（GLM4: 827.8）
4. 信号放大 = LayerNorm × 多层累积（每层1-3倍，36层=指数增长）

**类比**：LayerNorm 把房间（残差）缩小，但信号（Δ）没变，所以信号在房间中的"占比"变大了。

---

### 第四步：统计力学框架——信号传播方程

**怎么从实验推导出来的**：

1. P419：线性 Jacobian 只解释 1-27% 的变化 → **线性理论失败**
2. P442：三空间独立（cos≈0.02）→ **不存在确定性映射**
3. P439：recoding_ratio 恒定 → **统计均值稳定**

三个事实共同指向：信号传播是**统计性**的，不是确定性的。

```
信号传播方程：
  h_l = G_l · R_l · h_{l-1} + ε_l

  G_l = 逐层增益（LayerNorm主导）
  R_l = 随机旋转矩阵（cos→0, 方向逐渐偏离注入方向）
  ε_l = 噪声/非线性修正项

统计不变量：
  recoding_ratio = E[||P_{W_U}(h_l)||²/||h_l||²] ≈ 0.22
```

**类比**：类似热力学——温度（ratio=0.22）是统计量，单个分子运动（单层信号）不可预测，但宏观统计量稳定。

**P451 的颠覆**：PR 不是常数，是 SVD 截断 k 的函数！

| k | PR | PR/d |
|---|---|---|
| 300 | 219 | 0.086 |
| 560 | ~560 | ~0.22 ≈ 实测！ |
| 2558 | 2284 | 0.892 |

**结论**："2×因子"是 k=300 截断的数学伪影（truncation artifact）。

**但更深的问题**：W_U 几乎满秩（PR/d≈0.89），为什么 ratio 只有 0.22？
- 说明 ratio ≠ PR/d
- "有效覆盖维度"≠"能利用的维度"
- 需要新理论：**频谱耦合**——ratio = f(信号频谱, W_U频谱)

## W_U（Unembedding Matrix，反嵌入矩阵）

### 在 Transformer 中的位置

Transformer 语言模型的核心数据流：

```
输入文本 → Tokenizer → Token IDs → W_E (Embedding) → Transformer Layers → W_U (Unembedding) → Logits → 概率分布 → 下一个词
```

W_U 是模型的**最后一层**，负责把隐藏状态（hidden state）映射回词表空间。

### 具体定义

- **W_E**（Embedding Matrix）：shape = `[vocab_size, d_model]`，把 token ID 映射到 d_model 维向量
- **W_U**（Unembedding Matrix）：shape = `[vocab_size, d_model]`，把 d_model 维隐藏状态映射到 vocab_size 维 logits

```
logits = h_final @ W_U^T     # h_final: [1, d_model], W_U^T: [d_model, vocab_size]
                               # logits: [1, vocab_size]
```

W_U 的第 i 行就是词表中第 i 个词在 d_model 维空间中的"代表向量"。

### 在本项目中的关键意义

1. **W_U 行空间 = 语言可表达的空间**：W_U 有 151936 行（Qwen3 词表大小），但只有 2560 列（d_model），所以 151936 个词的向量全部挤在 2560 维空间中

2. **recoding_ratio 的定义**：信号 Δ（delta）在 W_U 行空间中的投影能量占比
   ```
   recoding_ratio = ||U^T · Δ||² / ||Δ||²
   ```
   其中 U 是 W_U 行空间的正交基（通过 SVD 得到）

3. **为什么研究 W_U**：如果一个信号在 W_U 行空间中无投影，那这个信号对输出词的预测**完全没有影响**。recoding_ratio≈0.22 意味着约 22% 的信号能量能影响输出，78% 被浪费了

4. **之前的关键发现**：
   - W_U 的 151936 行在 2560 维空间中**几乎正交**（cos≈0.004），这是 Johnson-Lindenstrauss 引理的体现
   - 词嵌入有数学结构（国王+王后-男性=女性），这些结构就编码在 W_U 的行向量中
   - W_U 接近满秩（PR/d≈0.89），说明 2560 维空间被词表向量几乎完全覆盖

```

### 第一步：L0→L1——非线性断裂层（整个网络最关键的一步）

### 2. "0.22 是高维稳态解" → ❌ **错误**

### 4. "随机旋转投影期望=0.22" → ❌ **需要修正**

纯随机旋转 → ratio(k) = k/d (指数=1.0 的幂律)

- 偏离很小 → 是"弱结构"旋转，不是"强聚焦"
- 真正的问题是：**什么样的弱结构旋转能产生 0.903 的幂律指数？**

### 5. "从牛顿力学到统计力学" → ⚠️ **类比方向对，但结论需要修正**

确实从"确定性动力学"转向"统计描述"，但：
- 不是简单的"随机旋转→统计不变量"
