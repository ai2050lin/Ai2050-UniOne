# AI2050 神经网络逆向工程系统（自动化执行版）

## 🎯 总目标

构建一个“自动科学家系统”，通过 AI 编程能力自动：

* 生成实验
* 运行分析
* 发现特性
* 验证因果
* 构建结构

最终实现：

> 从黑箱 Transformer → 可解释语义结构 + 动态生成机制

---

# 🧠 核心思想

系统建立在三元框架：

```
分析维度（What） × 分析操作（How） × 评估指标（Score）
```

并通过闭环实现自动进化：

```
假设 → 实验 → 结果 → 归纳 → 新假设
```

---

# 🧩 一、分析维度（What）

## 1. 词汇层（Lexical）

* token identity
* 多义性（apple vs Apple）

## 2. 句法层（Syntax）

* 依存关系
* 句法结构

## 3. 语义层（Semantics）

* 类别（fruit / object）
* 属性（color / size）
* 关系（is_a）

## 4. 概念层（Conceptual）

* 层级关系（fruit ⊂ food ⊂ object）
* 抽象程度

## 5. 上下文层（Contextual）

* 语义切换（apple → 公司 vs 水果）

## 6. 推理层（Reasoning）

* 传递关系
* 组合推理

## 7. 表示层（Representation）

* embedding
* hidden states
* attention

---

# ⚙️ 二、分析操作（How）

## 1. Probe（探针）

检测某信息是否存在

## 2. Ablation（消融）

删除组件测试必要性

## 3. Activation Patching

替换激活验证因果

## 4. Attribution（归因）

分析贡献来源

## 5. 子空间分析

PCA / SVD / clustering

## 6. 路径分析

信息流动路径

## 7. 相似度分析

cosine / CKA

---

# 📏 三、评估指标（Score）

每个实验必须输出：

* Separability（可分性）
* Causality（因果性）
* Stability（稳定性）
* Generalization（泛化性）
