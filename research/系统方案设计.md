## 零、核心思想与战略转变

### 0.1 项目本质

深度神经网络提取了大脑中的某种数学结构，产生了语言能力。语言能力既包括微观/中观/宏观多角度的知识体系，也包括语言本身的风格/逻辑/语法。这说明背后有某种数学结构——这个数学结构很可能超过了现在的数学体系，需要跳出预设理论，根据大量拼图进行分析和还原。

最终目标：

> 从黑箱 Transformer → 可解释语义结构 + 动态生成机制

建立：**神经网络内部科学（Neural Network Internal Science）**

### 0.2 455个Phase的核心教训

过去的455个Phase存在**三大根本问题**：

1. **没有系统性规划**：每次实验基于上一个"灵感"，导致30+拼图全是统计性质，功能机制几乎空白
2. **理论预设陷阱**：12条理论路线全部被推翻（ratio=0.22恒定量、频谱力学预测logit、平坦线性几何、ICSPB统一变量...），每次都是先预设理论再找证据
3. **统计量≠机制**：积累了大量cos/norm/probe accuracy等统计量，解释力从5-7%提升到18%，但深层机制仍待破解

### 0.3 战略转变：从"描述统计"到"逆向还原"

**旧路线（失败）**：观察统计现象 → 猜测理论 → 验证理论 → 被推翻
**新路线（当前）**：系统填充拼图 → 临界点自然涌现 → 从拼图还原数学结构

关键洞察：**不要猜理论，让数据说话。当拼图填充率>60%且3个统合格子被填充时，数学结构会自然涌现。**

## 四、核心未解问题与突破路径

### 问题1：深层为何能"锁定"浅层干预？★★★★★

**现象**：概念替换干预在浅层switch_rate=0.67-0.83，深层=0.00
**已知**：浅层确实编码概念信息（PCA5探针0.98），但深层对替换免疫
**最新进展**：语义力线指数增长(R3)表明深层信号量级远超浅层，可能是量级压制而非"纠错"
**突破路径**：
- 逐层追踪替换后的残差流变化，找到"纠正"发生的确切层
- 在该层分别消融注意力和FFN，观察是否恢复浅层干预的效果
- 检查深层norm增长(100-300x)是否直接压制浅层注入信号

### 问题2：中间层template热点的功能是什么？★★★★

**现象**：Qwen3=L18, GLM4=L20, DS7B=L14-21出现template主导的PC
**已知**：这些层template方差占比98.8%，可能与"句式处理"相关
**最新进展**：R16-R17表明语法角色编码在W_U⊥子空间，template热点可能是语法→W_U⊥编码的核心层
**突破路径**：
- 消融template热点层的template方向，观察输出是否失去语法结构
- 跨句式追踪template方向的共享度
- 验证template热点层的语法角色是否在W_U⊥中编码

### 问题3：知识×逻辑×语法的统一编码机制 ★★★★

**现象**："not"不否定属性方向，而是添加正交扰动（delta_attr_cos≈0）
**最新进展**：R16-R17揭示语法在W_U⊥编码，知识在W_U可见部分，逻辑在独立正交方向
**未知**：三者的子空间关系？容量分配？交互机制？
**突破路径**：
- SAE特征在知识/逻辑/语法任务中的复用率分析
- 测量W_U / W_U⊥中逻辑信号的占比
- 如果复用率>30%，说明存在统一基底（直接填充UN-1）

### 问题4：语法-语义解耦的完整机制 ★★★★★ (新, 5/3-5/4)

**现象**：语法角色在W_U⊥编码(lm_head看不到的方向)
**已知**：
- W_U⊥分类CV=100%, W_U分类CV=95-97%
- LDA方向能分类但因果效应≈0, 迁移方向因果效应最强
- 正四面体是LR artifact, 语法真实几何结构未知
**未知**：
- LayerNorm是否"泄露"W_U⊥信息到logits? (硬伤7)
- W_U⊥容量多大? 能容纳多少语法角色? (9C)
- 语法角色操控是否传播到最终输出? (9D)
**突破路径**：
- 绕过LayerNorm的操控实验(9B)
- W_U⊥中4个语法角色的余弦矩阵(9A)
- 中间层操控W_U⊥ → 追踪传播 → 测量logits变化(9D)

### 问题5：暗物质悖论 ★★★★ (升级)

**现象**：86-92%概念信号在W_U⊥(暗物质)，承载75-87%steering效果
**矛盾**：但W_U⊥操控KL仅为完整方向35-43%
**突破路径**：
- 非线性转导机制：暗物质如何"绕过"W_U到达logits？
- 暗物质级联转导：W_U注入6层后衰减到25-34%
- 概念传播 = 残差直通 + 非线性转导

### 问题5：从"统计描述"到"可预测理论" ★★★

**现象**：解释力只有5-7%
**已知**：大量统计量但无法从文本预测编码
**未知**：给定一个新句子，能否预测其残差流编码？
**突破路径**：先解决问题1-4，积累足够拼图后尝试构建预测方程

| Subspace(子空间) | 观察 | ✅ 成熟 | `pca(hidden, n_comp) → components, variance` |
| Attribution(归因) | 分析 | ⚠️ 部分可用 | `attribution(model, input, output) → contributions` |
| Ablation(消融) | 因果 | ⚠️ 需修复 | `ablate_head(layer, head) → modified_output` |
| Patching(替换) | 因果 | ✅ 成熟 | `patch_hidden(layer, tensor) → modified_output` |
| Path Tracing(路径) | 因果 | ❌ 未实现 | `trace_path(source, target) → causal_path` |

|------|------|---------|---------|
| Separability | 可分性 | probe accuracy, AUC | >0.8 (vs随机基线) |
| Causality | 因果性 | switch_rate, prob_lift | switch_rate>0.3 |
| Stability | 稳定性 | 跨模板/跨样本一致性 | cos>0.5 |

### 6.2 三种时间维度

## 十二、当前最大瓶颈与第一性原理分析

### 12.1 最大瓶颈：18%→60%的鸿沟

当前填充率18%，距离临界点60%仍有3倍差距。核心原因：

1. **暗物质悖论**：86-92%概念信号在W_U⊥，但W_U⊥操控KL仅35-43%，非线性转导机制不明
2. **知识网络三大硬骨头**：概念层次非嵌套(cos=-0.20) / 属性绑定完全未知 / 推理路径深层不成立
3. **语法真实几何未知**：正四面体被推翻(是LR artifact)，语法角色高维正交补空间结构待查
4. **三模型机制差异**：DS7B=Attn主导(21.5x), GLM4=Attn(5.07x), Qwen3=MLP主导(3.46x)

### 12.2 第一性原理：语言数学原理的四层框架

从逆向工程的角度，语言是一个**低维流形上的分布式代数动力系统**：

```
Language = Geometry + Algebra + Dynamics + Information Theory
```

#### Layer 1: 几何层 (Representation Geometry)

**问题**：信息位于哪些子空间？有效维度？流形结构？曲率？

**工具**：PCA/SVD/CCA/SVCCA/流形学习/曲率分析

**已有证据**：
- R6: 1-3个SVD模式控制97%输出
- R7: 增强对比(近义聚合/远义分离)
- R9: 旋转>90°=维度诅咒
- R13: 语义流形≈变形单纯形(fit_r2=0.87-0.96)
- R16: 语法在W_U⊥子空间

**核心地图**：子空间分解
```
R^d ≈ V_WU∩V_sem ⊕ V_WU⊥∩V_syn ⊕ V_WU⊥∩V_dark ⊕ V_⊥logic ⊕ V_residual
       语义(~14%)    语法(可分类)    暗物质(86-92%)   逻辑(独立)     余项
```

#### Layer 2: 代数层 (Algebraic Structure) ★ 新增

**问题**：子空间如何组合？是否存在群/半群/格/范畴结构？操作是否可交换？

**工具**：群表示论/格论/范畴论/线性代数

**已有线索**：
- R13: 变形单纯形结构 → 概念间的组合可能遵循单纯形规则
- R14: 同类正/异类负(cos=+0.08~-0.13) → 概念关系是"半序"而非"全序"
- R15: 传递性推理深层不成立 → 代数结构随层变化
- INV-13: "not"=正交扰动 → 逻辑操作可能形成群

**核心未解**：
- 语法变换群：nsubj↔dobj是否存在对合变换？
- 概念组合律：cat+wild是否=某方向加法？
- 逻辑操作代数：not/not是否恒等？and/or是否分配律？

#### Layer 3: 动力学层 (Computational Dynamics)

**问题**：信息如何在层间传播？哪些头负责写入/读取/转换？因果链？

**工具**：Activation patching/Causal tracing/Jacobian/动态系统分析

**已有证据**：
- INV-6: 加法传播定律(gap(l+1)=gap(l)+delta(l), r=1.000)
- R3: 语义力线指数增长(exp(0.17~0.25*l), 100-300x)
- R4: Attention是语义→Logit主通道(21.5x vs MLP 1.87x)
- R1: FFN增益调制×内容方向分解
- R8: 三组件几何角色(残差锚+Attn交换+MLP反射)

**核心未解**：
- 暗物质转导：W_U⊥信号如何"绕过"W_U到达logits？
- 深层锁定：语义力线指数增长(100-300x)是否压制浅层注入？
- 注意力头的精确角色：哪些头写入V_sem？哪些写入V_syn？

#### Layer 4: 信息论层 (Information Allocation)

**问题**：信息如何分配？为何冗余编码？显式/隐式分离的原理？

**工具**：Mutual Information/Fisher Information/信息瓶颈/谱分析

**已有证据**：
- R12: 86-92%概念信号在W_U⊥(暗物质), 承载75-87%steering效果
- R16: 语法"隐身"在W_U⊥(lm_head看不到的方向) — **信息隐藏策略**
- R2: W_down压缩掉z空间特异性 — **信息瓶颈**
- 硬伤5: 暗物质悖论(86-92%范数但操控KL仅35-43%)

**核心假说**：
- 信息瓶颈原则：W_down压缩是信息瓶颈的实例
- 最优编码假说：W_U⊥语法编码是最优球面编码的实例
- 冗余编码原则：分布式编码提升鲁棒性(硬伤1: 任何方向perturb都有效)

#### 四层框架的统一命题

> 语言是一个在高维神经空间中运行的**低维流形上的分布式代数动力系统**。
> 
> 其核心特征：
> 1. **低维流形**：承载抽象结构(R6: 3个SVD模式→97%)
> 2. **代数操作**：实现组合规则(R13: 单纯形/R14: 半序)
> 3. **动力传播**：实现逐层计算(R3: 指数增长/R4: Attn主通道)
> 4. **冗余编码**：提升鲁棒性(R12: 暗物质/R16: 信息隐藏)
> 5. **可读写分离**：实现控制与生成解耦(R17: LDA可读但因果≈0)

### 12.3 突破瓶颈的关键洞察

**核心洞察**：语言是低维流形上的分布式代数动力系统。四层框架(几何+代数+动力学+信息论)中，**几何层和数据动力学层已有18条规律支撑，但代数层和信息论层几乎空白**。

当前最大的三个突破口：

1. **语法-语义解耦(R16-R18)**：语法在W_U⊥编码，是语言模型的"信息隐藏"策略。W_U⊥→logits转导机制=信息论层的核心问题
2. **暗物质悖论(R12)**：86-92%概念信号在W_U⊥但操控KL仅35-43%，非线性转导=动力学层的核心问题
3. **概念组合的代数结构(R13-R15)**：变形单纯形+传递性崩塌+非嵌套层级=代数层的核心问题

**关键教训**：正四面体是LR artifact(R18)——几何结构可能被分类器偏差扭曲。所有几何发现必须通过因果验证(铁律3)。

**从"统计描述"到"统一理论"的路径**：
```
当前(18%填充率) → 代数层实验(构造变换群/组合律) → 信息论层实验(信息瓶颈/容量)
                  → 四层统一验证(跨模型/跨语言/因果性)
                  → 临界点(60%填充率+3个UN格子)
                  → 语言统一数学理论
```
